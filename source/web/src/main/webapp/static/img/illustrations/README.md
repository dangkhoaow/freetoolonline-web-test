## Editing

Do NOT hand-edit these files. They are generated by
`.agent/skills/seo-svg-diagram-author/scripts/render-svg.mjs` with a
deterministic param-hash. Hand-edits get overwritten on the next render.

To regenerate: re-run `render-svg.mjs` with the same params (no-op if file
exists with identical content), or with different params (produces a NEW
filename — the old file remains as an orphan until Phase-2 cleanup).

## File naming convention

```
<template-id>/<context-slug>__<hash-8>.svg
```

Where `<hash-8>` is `sha256(JSON.stringify(params)).slice(0,8)`. See
`.agent/skills/seo-svg-diagram-author/references/asset-naming-convention.md`.
