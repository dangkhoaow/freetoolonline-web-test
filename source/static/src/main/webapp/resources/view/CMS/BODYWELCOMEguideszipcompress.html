Name the archive before you download it. In the Settings panel, type a filename that reflects the project, the client, or the date - "myzipped-<date>" is the default but a name like "annual-report-2026" tells the recipient what they are opening before they unzip it. If the file list is empty when you press the Zip button, the page surfaces an inline error in the download row instead of producing a blank archive; re-add the files and try again. Small details like these turn a generic .zip into a bundle the recipient can read at a glance.
