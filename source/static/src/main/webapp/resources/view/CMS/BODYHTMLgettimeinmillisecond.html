When to use the current time in milliseconds

Use this Unix-epoch readout when a piece of code, a log line, an API request, or a test fixture needs the current time as a single integer. The value updates every second, so you can copy it straight into a JSON payload, a database column, or a debugger watch. The page shows the current time in your local timezone alongside the millisecond figure, which makes it easy to sanity-check a server-side timestamp without converting in your head. If you need the reverse direction - turning a millisecond integer back into a date - pair this readout with Convert time in millisecond to date. Free, in-browser.
