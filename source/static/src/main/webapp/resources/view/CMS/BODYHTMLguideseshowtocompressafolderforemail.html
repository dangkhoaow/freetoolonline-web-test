Como Comprimir una Carpeta para Correo - Paso a Paso (2026)

Ultima revision 2026-04-24. Funciona en Windows 10/11, macOS 13+ (Ventura y posterior), y en el navegador via Zip File cuando un flujo zip-shell nativo no esta disponible.

Respuesta de 30 segundos. Haz clic derecho en la carpeta → "Enviar a carpeta ZIP (comprimida)" en Windows (o "Comprimir" en macOS). Adjunta el .zip resultante al correo. Si el ZIP excede el limite del proveedor (tipicamente 20-25 MB, ver tabla de tamano de correo abajo), entonces (a) divide la fuente en dos carpetas y comprime cada una, o (b) sube a un servicio de transferencia de archivos y pega el enlace en el cuerpo del correo.

Por que comprimir una carpeta para correo?

Los adjuntos de correo se miden en bytes. Las carpetas no comprimidas no pueden ser adjuntadas - los clientes de correo requieren un solo archivo, no un directorio. Comprimir colapsa la carpeta en un solo archivo ZIP que preserva la estructura interna, nombres de archivos y fechas de modificacion, mientras encoge el tamano total a traves de compresion LZ77 / DEFLATE. Para carpetas de contenido mixto (documentos, imagenes, hojas de calculo), espera reduccion de tamano del 20-50%; para carpetas ya dominadas por formatos comprimidos (JPG, MP3, MP4), espera solo unos pocos porcentajes porque esos formatos ya estan codificados por entropia.

Las tres razones practicas para comprimir antes de enviar:

  • Limites de tamano del proveedor. Gmail, Outlook e iCloud todos limitan los adjuntos individuales. Un ZIP puede caber bajo el limite cuando la carpeta cruda no puede.
  • Preservacion de estructura. El destinatario obtiene el arbol de carpeta intacto despues de descomprimir - no 47 archivos sueltos en la raiz de su carpeta Descargas.
  • Integridad en transito. ZIP almacena CRC-32 por archivo; descargas corruptas se senalizan al extraer para que puedas reintentar en lugar de recibir silenciosamente bytes rotos.

Paso a paso en Windows 10 y 11

  1. Localiza la carpeta en el Explorador de Archivos. Si estas comprimiendo un subconjunto de archivos en lugar de la carpeta entera, crea una nueva carpeta vacia y copia los archivos a ella primero - ZIP no puede "seleccionar-y-archivar" archivos sueltos nativamente sin agruparlos.
  2. Haz clic derecho en la carpeta. En Windows 11 aparece el nuevo menu shell; si no ves la opcion ZIP inmediatamente, haz clic en "Mostrar mas opciones" en la parte inferior para revelar el menu clasico.
  3. Elige "Enviar a" → "Carpeta comprimida (zipada)". Windows crea <nombre-de-carpeta>.zip en el mismo directorio.
  4. Renombra el ZIP si lo deseas (el predeterminado hereda el nombre de la carpeta).
  5. Adjunta el ZIP a tu correo. Arrastra-y-suelta en la ventana de composicion o usa el boton de clip / "Adjuntar archivo".

Windows 11 agrego soporte nativo de 7z y tar en 2023; para adjuntos ZIP especificamente, quedate con .zip porque macOS y cada panel de vista previa de webmail lo decodifican sin descarga. Si el destinatario confirma que 7-Zip o equivalente esta instalado, cambiar a .7z con LZMA2 puede recortar otro 10-20% del tamano al costo de un paso de compresion mas lento.

Paso a paso en macOS 13+ (Ventura, Sonoma, Sequoia)

  1. Selecciona la carpeta en Finder.
  2. Haz clic derecho (o Control-clic) → "Comprimir '<nombre-de-carpeta>'". macOS crea <nombre-de-carpeta>.zip en el mismo directorio.
  3. Renombra el archivo si es necesario. macOS preserva metadatos de archivo y resource forks dentro de carpetas __MACOSX, que algunos destinatarios Windows consideran desorden; ve el FAQ sobre como eliminarlos abajo.
  4. Adjunta al correo via arrastrar-soltar estandar o accion Adjuntar Archivo.

macOS tambien soporta archivado basado en Terminal con zip -r archive.zip carpeta/ (BSD zip) o ditto -c -k --sequesterRsrc carpeta archive.zip (comando ditto de Apple). Ditto es lo que Finder invoca bajo el capo y es preferible para cualquier carpeta que contenga resource forks especificos de macOS.

Limites de adjuntos de correo que bloquean una carpeta ZIPada en 2026

Los limites especificos del proveedor causan la mayoria de errores "mi ZIP de carpeta no envia". El bloqueador casi siempre es el techo del destinatario, no tu limite de envio - conoce ambos antes de comprimir, porque un ZIP de carpeta de 30 MB que tu Gmail envia bien aun rebota contra un destinatario Outlook de 25 MB. Las carpetas son particularmente afectadas porque los archivos de directorio tienden a empaquetar docenas de elementos a la vez; una foto sobredimensionada dentro de la carpeta es suficiente para empujar el ZIP entero sobre el limite de un destinatario.

  • Gmail: envio 25 MB / recibo 50 MB. Adjuntos mas grandes se auto-redirigen a traves de Google Drive.
  • Outlook (Microsoft 365): envio predeterminado 20 MB; recibo 25 MB para la mayoria de planes. Inquilinos empresariales pueden configurarse hasta 150 MB pero destinatarios fuera del inquilino a menudo aun ven 25 MB.
  • Outlook.com (gratis): limite de envio de 10 MB antes del redireccionamiento OneDrive.
  • iCloud Mail: 20 MB; Mail Drop de Apple entra en accion para adjuntos mas grandes (servidor iCloud aloja el archivo por 30 dias).
  • Yahoo Mail: 25 MB.
  • SMTP corporativo / personalizado: varia mucho (10-150 MB). Pregunta a tu administrador o haz un envio de prueba con un archivo de ~15 MB.

Cuando el ZIP excede el limite del destinatario, el mensaje de rebote generalmente dice "mensaje demasiado grande" o "554 5.7.1." Si ves ese error, no reenvies con el mismo ZIP - cambia a una estrategia de enlace de transferencia de archivo.

Cuando agregar una contrasena al ZIP

Anade una contrasena cuando la carpeta contenga cualquiera de: documentos legales, contratos firmados, datos de RR.HH., registros financieros, credenciales de login, o cualquier cosa sujeta a un acuerdo de confidencialidad. ZIP protegido por contrasena cifra el contenido del archivo con AES-256 (el cifrador "tradicional" ZIP 2.0 esta roto y no debe usarse). El destinatario necesita la contrasena via un canal separado - no pongas la contrasena en el mismo correo que el adjunto.

El camino mas rapido en el navegador es Zip File que crea ZIPs protegidos por contrasena sin instalacion de software. Si solo necesitas abrir un ZIP ya protegido por contrasena que controlas, la herramienta correspondiente es Quitar Contrasena ZIP. Para la matriz corta de decision sobre quien debe saber que, ve la guia relacionada sobre tipos de contrasena PDF: propietario vs usuario - la misma taxonomia aplica a permisos de acceso vs edicion ZIP.

Cuando saltar ZIP y usar un servicio de transferencia de archivos

ZIP es la respuesta correcta para carpetas bajo el techo de adjuntos del destinatario. Encima de eso, los flujos de trabajo basados en adjuntos se vuelven fragiles - los destinatarios rebotan, los filtros de spam senalan, las cuotas corporativas de bandeja de entrada se llenan. Tres senales de que un servicio de transferencia de archivos (Google Drive, OneDrive, Dropbox, WeTransfer, o tu propio hosting) supera un adjunto:

  • ZIP mas grande que 20 MB yendo a cualquier cuenta de correo de consumidor.
  • Destinatarios en correo corporativo restrictivo (legal, financiero, salud frecuentemente tienen filtros agresivos de tamano y contenido).
  • Carpetas ricas en medios (video, fotos raw) donde la compresion no reducira el tamano significativamente - hospedar es mas eficiente que adjuntar.

Si necesitas que el destinatario pueda ver el contenido sin descargar primero (revision de solo lectura), un enlace compartible a una carpeta en la nube supera a un adjunto ZIP en cada eje: previsualizable, siempre actualizado, e individualmente revocable.

Reduciendo el tamano del ZIP antes de enviar

Si tu carpeta comprime a justo encima del limite de tamano, cuatro reducciones pragmaticas antes de cambiar a transferencia de archivo:

  • Comprime imagenes primero. JPG en calidad 85 es usualmente indistinguible de la fuente y 30-50% mas pequeno. Ejecuta las imagenes sueltas a traves de Compress Image antes de comprimir.
  • Convierte fotos HEIC del iPhone. Archivos HEIC, una vez dentro de un ZIP, no pueden ser previsualizados en muchos destinatarios Windows. Convierte via HEIC to JPG primero - JPG es mas grande por archivo pero abre en todas partes sin una instalacion de codec. Ve HEIC vs JPG vs WebP para la decision completa de formato.
  • Elimina metadatos macOS. En macOS, los archivos Finder incluyen ruido __MACOSX y .DS_Store. El comando ditto los quita; alternativamente, comprime en una maquina Linux / Windows que nunca los produce.
  • Divide en multiples ZIPs. Muchos clientes de correo aceptan 2-3 adjuntos mas graciosamente que un ZIP sobredimensionado. Agrupa por tema (documentos.zip + fotos.zip) en lugar de por limite de byte arbitrario.

Preguntas Frecuentes

Por que mi ZIP sigue excediendo el limite despues de la compresion?

Porque la carpeta fuente esta dominada por contenido ya comprimido. JPG, PNG, MP3, MP4, DOCX y XLSX estan todos pre-comprimidos internamente; el algoritmo DEFLATE de ZIP tiene poca entropia adicional para eliminar. Para carpetas donde 80%+ de los bytes vienen de estos formatos, espera solo unos pocos porcentajes de reduccion. En esos casos, cambia a un servicio de transferencia de archivos en lugar de tratar de exprimir el ZIP bajo el limite.

Puedo proteger con contrasena un ZIP de carpeta en Windows sin software extra?

El comando "Enviar a → Carpeta comprimida" del shell Windows no soporta proteccion por contrasena nativamente. Tienes tres opciones: (a) usa nuestra herramienta basada en navegador Zip File para crear un ZIP protegido por contrasena, (b) instala 7-Zip y elige "Anadir al archivo..." con una contrasena, o (c) usa PowerShell con el cmdlet Compress-Archive mas un paso de cifrado de terceros. La herramienta de navegador es la opcion de menor friccion porque no se requiere instalacion.

Que pasa si el destinatario dice "el ZIP esta corrupto" o "la extraccion fallo"?

Tres causas comunes: (1) el escaner antivirus del proveedor de correo modifico el archivo en transito (inusual pero pasa con gateways antivirus mas antiguos); (2) el ZIP fue creado con Windows Explorer y contiene nombres de archivos no-ASCII que el extractor del destinatario no puede decodificar; (3) la descarga fue truncada (menos probable en HTTP moderno pero aun posible en redes moviles inestables). Arreglar: re-comprime la carpeta en un SO diferente (si fue creado en macOS, intenta de nuevo en Windows o usa ditto con --sequesterRsrc), o re-sube via un servicio de transferencia de archivos que verifica checksums del lado del servidor antes de exponer el enlace de descarga.

Resumen

Comprimir una carpeta para correo es una operacion de shell nativo de 10 segundos en Windows y macOS. La decision real es si el ZIP resultante encaja bajo el techo de adjuntos del destinatario - si si, enviala con el menu shell de clic derecho; si no, elige entre dividir en ZIPs mas pequenos y cambiar a un enlace de transferencia de archivo. Anade una contrasena cuando la carpeta contenga cualquier cosa regulada o confidencial, y recuerda que HEIC y medios ya comprimidos no encogeran mucho - convierte o aloja esos separadamente.

Listo para crear tu ZIP? Comienza con Zip File para compresion basada en navegador con proteccion opcional por contrasena. Para la direccion inversa, la herramienta Quitar Contrasena ZIP desbloquea archivos que ya tienes permiso para abrir.
