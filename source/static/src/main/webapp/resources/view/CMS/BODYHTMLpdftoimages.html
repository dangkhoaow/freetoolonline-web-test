What this tool does: rasterizes each page of the uploaded PDF into a separate PNG or JPG image at the output DPI you select (72, 150, or 300). Each page becomes one image file, packaged together so you can download them as a batch.

What it does NOT do: no OCR (text inside the image stays as pixels, not editable text); no text extraction (use PDF to Text for that); no edits to the source PDF (the original returns unchanged).
