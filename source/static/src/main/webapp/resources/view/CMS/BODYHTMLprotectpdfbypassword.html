What this tool does: takes a PDF you can already open and attaches an open password (also called the user password). The output is a copy of the same PDF that prompts every subsequent reader for the password you set here before any page renders.

What it does NOT do: it does not modify permissions on an already-encrypted PDF, and it does not unlock a file. To remove an existing password use /remove-pdf-password.html.
