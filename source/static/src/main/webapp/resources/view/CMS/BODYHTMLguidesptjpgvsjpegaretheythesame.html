JPG vs JPEG - Sao o Mesmo?

Ultima revisao: 2026-04-27. Use HEIC para JPG ou Comprimir imagem para qualquer trabalho com fotos no seu navegador.

Resposta de 30 segundos. Sim - JPG e JPEG sao o mesmo formato de imagem. Ambas as extensoes apontam para o mesmo padrao (ISO/IEC 10918, o formato do Joint Photographic Experts Group de 1992). As duas extensoes existem porque o Windows antes de 95 limitava extensoes a tres letras, entao .jpeg foi truncado para .jpg.

A historia em um paragrafo

O formato e nomeado pelo comite de padroes que o definiu - o Joint Photographic Experts Group, JPEG. A extensao natural era .jpeg. Mas MS-DOS e Windows 3.x usavam o sistema de arquivos FAT16, que limitava extensoes a tres caracteres. Para permanecer legivel, a extensao foi encurtada para .jpg.

O que esta de fato no arquivo

Tanto arquivos .jpg quanto .jpeg contem a mesma sequencia de bytes: um cabecalho magic-number JPEG (os bytes FF D8 FF), dados de imagem codificados, e metadados. Renomeie um .jpg para .jpeg - ainda abre. Os bytes nao se importam.
