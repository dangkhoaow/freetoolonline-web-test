Mengapa HEIC Tidak Bisa Dibuka di Windows - Tiga Solusi Cepat

Ditinjau terakhir 2026-05-04. Seorang teman mengirim foto dari iPhone-nya melalui AirDrop atau email, Anda mengeklik ganda file .heic di PC Windows, dan Windows menjawab dengan "Windows tidak bisa membuka file ini" atau hanya menampilkan ikon generik dan menolak untuk pratinjau. Alasannya sederhana: HEIC adalah format foto default di iPhone sejak iOS 11 (2017), tetapi Windows tidak menyertakan codec yang diperlukan untuk mendekodekannya secara bawaan. Panduan ini menjelaskan tiga solusi yang mencakup hampir setiap laporan "HEIC tidak bisa dibuka di Windows" - pasang codec, gunakan penampil pihak ketiga gratis, atau konversi file ke JPG online dengan alat gratis HEIC to JPG - dan memberi tahu Anda solusi mana yang menang untuk situasi mana.

Keputusan 30 detik. Jika Anda hanya perlu melihat SATU foto dan tidak ingin memasang apa pun, konversi secara online dengan HEIC to JPG - unggah, unduh JPG, selesai. Jika Anda secara rutin menerima foto HEIC dari teman pengguna iPhone dan ingin foto itu "berjalan begitu saja" di aplikasi Photos Windows, pasang HEIF Image Extensions gratis dari Microsoft Store (Solusi 1). Jika Anda tidak memiliki hak admin atau tidak bisa memasang ekstensi Store di PC kerja yang dikelola, gunakan penampil portabel pihak ketiga seperti IrfanView dengan plugin HEIF (Solusi 2). Tiga solusi mencakup kasus sekali pakai, berulang, dan PC terkunci masing-masing; tabel perbandingan di dekat bagian bawah halaman ini memetakan setiap kasus ke pilihan yang tepat.

Mengapa ini terjadi

Apple mengubah format foto default di iPhone dari JPG ke HEIC (High Efficiency Image Container) mulai dari iOS 11 pada 2017. Manfaat utamanya adalah sekitar setengah ukuran file untuk kualitas visual yang sama, yang penting ketika iPhone menyimpan puluhan ribu foto. Kompromi adalah dukungan platform: HEIC bergantung pada dekoder HEVC (H.265), yang Microsoft lisensikan secara terpisah dan secara historis tidak menyertakan dengan Windows. Instalasi standar Windows 10 atau Windows 11 oleh karena itu tahu apa itu file .heic - File Explorer menunjukkan ekstensi - tetapi tidak bisa mendekodekan pixel di dalamnya, sehingga aplikasi Photos menampilkan ikon placeholder atau kesalahan.

Kabar baiknya adalah bahwa ini adalah masalah codec, bukan masalah perangkat keras. Setelah codec dipasang (atau setelah file dikonversi ke format yang sudah dipahami Windows, seperti JPG), setiap aplikasi Windows yang menggunakan tumpukan gambar sistem - Photos, thumbnail File Explorer, Word, PowerPoint, lampiran Outlook, drag-and-drop browser - menampilkan foto tanpa konfigurasi lebih lanjut. Ada beberapa jalur untuk "memiliki codec yang tersedia", dan yang tepat tergantung pada apakah Anda menggunakan Windows 10 atau 11, apakah Anda memiliki hak admin, dan apakah Anda membutuhkan ini untuk satu foto atau untuk aliran gambar berkelanjutan yang bersumber dari iPhone. Lihat juga panduan terkait tentang format foto iPhone dijelaskan (HEIC, JPG, PNG, RAW) untuk sisi format.

Solusi 1 - pasang HEIF Image Extensions di Windows 10 / 11

Jalur resmi adalah memasang HEIF Image Extensions Microsoft dari Microsoft Store. Ini menambahkan dekoder foto HEIF ke tumpukan imaging Windows sehingga aplikasi Photos, File Explorer, dan konsumen gambar sistem lainnya dapat merender file .heic sebaris. Ekstensi itu sendiri gratis; pada sebagian besar instalasi Windows 11, ekstensi video HEVC yang dikirim terpisah juga sudah terpasang, sehingga jalur foto bekerja secara bawaan setelah ekstensi HEIF ditambahkan. Pada Windows 10, Anda mungkin juga perlu memasang paket HEVC Video Extensions dari Store yang sama (paket ini saat ini memiliki biaya sekali kecil dari Microsoft untuk lisensi codec video) sebelum foto HEIC dirender dengan benar.

Langkah-langkah:

  • Buka aplikasi Microsoft Store dari menu Start.
  • Cari HEIF Image Extensions (penerbit: Microsoft Corporation).
  • Klik Get dan biarkan terpasang. Unduhan kecil (di bawah beberapa megabyte) dan tidak memerlukan restart.
  • Klik kanan pada file .heic - Open with - Photos. Gambar sekarang harus dirender sebaris.
  • Jika foto masih belum dirender pada Windows 10, pasang juga HEVC Video Extensions dari Store (catatan: paket ini berbayar; build "Device Manufacturer" gratis hanya tersedia di PC yang produsennya melisensikannya sebelumnya).

Solusi ini adalah jawaban yang tepat untuk pengguna PC rumah mana pun yang secara rutin menerima foto dari teman pengguna iPhone dan ingin aplikasi Photos dan thumbnail File Explorer "berjalan begitu saja" mulai sekarang. TIDAK bekerja pada PC kerja terkunci di mana akses ke Microsoft Store diblokir oleh kebijakan grup - jika Store dinonaktifkan untuk Anda, lompat ke Solusi 2 atau Solusi 3.

Solusi 2 - gunakan penampil pihak ketiga gratis (CopyTrans HEIC, IrfanView dengan plugin HEIF)

Jika Anda tidak bisa memasang ekstensi Store - PC kerja terkunci, tanpa hak admin, komputer sekolah, kios, versi Windows yang lebih lama yang tidak memiliki Store - penampil pihak ketiga melewati tumpukan codec sistem sepenuhnya. Dua opsi gratis yang sudah mapan:

  • CopyTrans HEIC untuk Windows. Unduhan gratis kecil dari CopyTrans yang menambahkan dukungan thumbnail dan pratinjau HEIC ke Windows Explorer dan Windows Photo Viewer bawaan. Setelah terpasang, mengeklik ganda file .heic membukanya di Photo Viewer dengan cara yang sama seperti JPG. Membutuhkan hak admin untuk memasang (ini ekstensi shell level sistem), jadi tidak menyelesaikan kasus PC kerja terkunci.
  • IrfanView dengan paket plugin HEIF. IrfanView adalah penampil gambar Windows gratis yang sudah lama berdiri yang mendukung mode portabel (tanpa installer, berjalan dari stik USB atau folder di direktori home Anda). Unduh IrfanView dan paket plugin yang cocok dari situs resmi irfanview.com, taruh di folder, dan buka file .heic melalui UI IrfanView. Mode portabel bekerja pada PC terkunci karena tidak menulis ke folder sistem atau registri Windows.

Kompromi: penampil pihak ketiga hanya memperbaiki tampilan HEIC di dalam UI mereka sendiri. Aplikasi Photos Windows, thumbnail File Explorer, Word, dan PowerPoint masih akan melihat .heic sebagai format yang tidak dikenal kecuali Anda juga memasang Solusi 1. Untuk "saya hanya perlu melihat satu foto ini sekarang" pada PC di mana Anda tidak bisa memasang codec di seluruh sistem, IrfanView portabel adalah jawaban paling bersih.

Solusi 3 - konversi HEIC ke JPG online untuk foto sekali pakai

Jika Anda tidak ingin memasang apa pun sama sekali, jalur tercepat adalah mengonversi file HEIC ke JPG sekali dan kemudian membuka JPG dengan aplikasi Photos, Paint, atau alat Windows lain yang selalu menangani JPG. Alat gratis HEIC to JPG menjalankan konversi di sisi server: Anda mengunggah file .heic, konversi berjalan pada layanan AWS freetoolonline, dan Anda mengunduh JPG. File asli dirotasi keluar dari penyimpanan layanan setelah jendela retensi standar, jadi Anda harus mengunduh dan menyimpan JPG secara lokal jika ingin menyimpannya.

Ini adalah solusi yang tepat ketika:

  • Anda hanya perlu melihat (atau berbagi dengan teman non-iPhone, atau melampirkan ke email) SATU foto, bukan aliran foto.
  • Anda berada di PC terkunci di mana Anda tidak bisa memasang codec atau penampil pihak ketiga, tetapi Anda bisa menggunakan browser web.
  • Anda menginginkan foto dalam JPG karena tujuannya (sebuah situs web, CMS, percetakan) tidak menerima HEIC.
  • Anda perlu berbagi foto dengan seseorang yang perangkatnya tidak mendukung HEIC (ponsel Android yang lebih lama, desktop Linux, versi macOS yang lebih lama).

Jika Anda memiliki folder berisi 50+ foto HEIC untuk dikonversi sekaligus, lihat Cara mengonversi 100 foto HEIC ke JPG untuk alur batch - alat /heic-to-jpg.html yang sama menangani input batch. Untuk satu foto atau beberapa, alur unggah-unduh sederhana adalah jalur tercepat.

FAQ

Apakah paket HEIF Image Extensions gratis di Windows 11? Ya - ekstensi foto itu sendiri gratis di Microsoft Store. Pada Windows 10, paket HEVC Video Extensions yang cocok (diperlukan untuk beberapa file HEIC) saat ini memiliki biaya sekali kecil dari Microsoft untuk lisensi codec video, meskipun banyak PC Windows 10 yang produsennya melisensikan HEVC sebelumnya mendapatkan build "Device Manufacturer" gratis secara otomatis.

Bisakah saya hanya mengganti nama .heic menjadi .jpg dan membuat Windows membukanya? Tidak. Ekstensi file tidak mengubah konten sebenarnya dari file - file HEIC yang diganti namanya menjadi photo.jpg tetap merupakan file HEIC, dan Windows masih tidak bisa mendekodekannya tanpa codec. Mengganti nama akan menipu File Explorer untuk mencoba membuka file dengan handler JPG, tetapi handler akan gagal karena byte di dalamnya bukan JPG. Gunakan Solusi 3 (konversi sesungguhnya) sebagai gantinya.

Apakah mengonversi ke JPG akan kehilangan kualitas? Ya, sedikit. HEIC menyimpan foto dengan kompresi HEVC; JPG menggunakan codec yang lebih lama dan kurang efisien. Bolak-balik dari HEIC ke JPG menghasilkan kehilangan kualitas yang terlihat hanya di bawah inspeksi pixel-by-pixel untuk sebagian besar foto. Untuk melihat, berbagi, dan mencetak, kehilangan dapat diabaikan. Jika Anda membutuhkan kualitas HEIC asli, simpan file HEIC di samping JPG - konversi menghasilkan file baru dan tidak memodifikasi sumber. Lihat HEIC vs JPG vs WebP untuk perbandingan format berdampingan.

Apakah pengaturan "Most Compatible" di sisi iPhone memperbaiki ini di sumbernya? Ya - di iPhone, Settings - Camera - Formats - Most Compatible mengganti foto baru ke JPG alih-alih HEIC. Ini memperbaiki foto mendatang dari iPhone tertentu itu tetapi bukan foto HEIC yang sudah diambil. Untuk foto HEIC yang sudah ada di Camera Roll, konversi (Solusi 3) adalah satu-satunya jalur.

Apa yang BUKAN panduan ini

Dua tugas pembaca yang berdekatan sengaja di luar cakupan:

  • "Bagaimana saya membuka HEIC di macOS / Linux / Android?" macOS membuka HEIC secara native (ini adalah format Apple sendiri). Distribusi Linux bervariasi - sebagian besar build desktop modern menyertakan libheif dan dapat pratinjau HEIC di pengelola file, tetapi build yang lebih lama tidak. Android 10+ mendukung HEIC secara native; Android yang lebih lama membutuhkan aplikasi galeri pihak ketiga. Panduan ini terbatas pada Windows.
  • "File HEIC saya terbuka tetapi terlihat salah (terputar, warna pudar, transparansi hilang)." Itu adalah diagnostik yang berbeda - codec berfungsi tetapi output tidak cocok dengan apa yang ditampilkan iPhone. Metadata orientasi sensor, profil warna wide-gamut, dan data gerakan Live Photo adalah tersangka biasa; panduan mendatang akan mencakup kasus itu.

Panduan pendamping untuk kategori HEIC / konversi gambar: HEIC vs JPG vs WebP - kapan menggunakan yang mana (perbandingan format), Cara mengonversi HEIC ke JPG langkah demi langkah (alur konversi), Cara mengonversi 100 foto HEIC ke JPG (alur batch), dan format foto iPhone dijelaskan (HEIC vs JPG vs PNG vs RAW di iPhone).
