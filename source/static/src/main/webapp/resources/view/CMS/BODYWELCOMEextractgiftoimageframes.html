FAQ

Is this a gif to frames, gif to images, or gif to png/jpg tool?
All four. Each animated GIF frame becomes its own still image, and you choose PNG (lossless, supports transparency) or JPG (smaller file size, no transparency) per export. Whichever phrasing you searched for — "gif to frames", "gif to images", "convert gif to png", "convert gif to jpg" — the same upload + extract action produces the file set.

What does "FPS" do when I extract frames?
FPS controls how many frames per second of the source GIF the extractor saves. A lower FPS exports fewer frames (smaller download, more spaced-apart stills), a higher FPS exports more frames closer together. The page GIF frames vs GIF frame rate (FPS) explained walks through which FPS to pick for the most common tasks (hero frame for a social post, animation key-frames, slow-motion still capture).
