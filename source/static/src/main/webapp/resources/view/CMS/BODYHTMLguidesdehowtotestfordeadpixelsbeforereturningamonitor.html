So testen Sie tote Pixel bevor Sie einen Monitor zurueckgeben

Veroeffentlicht 20. April 2026

Ein neuer Monitor mit einem toten Pixel ist aergerlich; ein toter Pixel, der eine Woche nach Ablauf des Rueckgabefensters entdeckt wird, ist teuer. Die meisten Einzelhaendler und Hersteller erlauben eine Rueckgabe oder einen Umtausch innerhalb eines angegebenen Fensters, aber nur wenn der Fehler verifiziert ist. Dieser Leitfaden geht den 5-minuetigen Test durch, der tote Pixel, festsitzende Pixel, Hintergrundbeleuchtungsblutung und Farbgleichmaessigkeitsprobleme erkennt, bevor Sie ueber den Rueckgabe-Termin hinaus sind - und deckt ab, welche Dokumentation Sie benoetigen, wenn die Rueckgabe zur Garantie-Pruefung geht.


Toter Pixel vs festsitzender Pixel vs Sub-Pixel

Toter Pixel. Ein einzelner Pixel, der nie leuchtet. Auf einem weissen Hintergrund erscheint er schwarz; auf jedem farbigen Hintergrund erscheint er ebenfalls schwarz. Der Transistor, der diesen Pixel ansteuert, ist ausgefallen; nichts, was Sie an der Software tun, wird das beheben. Dies ist der Fehler, den Hersteller als Rueckgabe akzeptieren.

Festsitzender Pixel. Ein Pixel, der immer eine Farbe zeigt, unabhaengig davon, was das Betriebssystem zeichnet. Normalerweise rot, gruen oder blau festgesetzt, weil ein Sub-Pixel permanent "an" ist. Manchmal behebbar durch Ausfuehren einer Pixel-Zyklus-Software ("JScreenFix"-Muster, FFmpeg pixel-flicker) fuer einige Stunden; haeufig nicht. Die meisten Hersteller akzeptieren festsitzende Pixel nicht als Rueckgabe, es sei denn, Sie haben mehrere gruppiert.

Sub-Pixel-Defekt. Einer der drei Sub-Pixel (R, G oder B) in einem einzelnen Pixel ist festgesetzt oder tot. Schwerer zu sehen - zeigt sich als leichter Farbton auf einheitlichen Farben. Im Allgemeinen kein Grund zur Rueckgabe.


Die 5-Minuten-Testsequenz

Verwenden Sie unser LCD-Test-Tool im Vollbildmodus auf dem zu testenden Display. Das Tool zyklisiert solide Farben - rot, gruen, blau, schwarz, weiss, gelb - jeweils eine pro Vollbild.

  1. Dunkeln Sie den Raum ab. Umgebungslicht maskiert subtile Defekte. Ein dunkler Raum nachts ist ideal; Jalousien schliessen und Deckenlichter dimmen funktioniert tagsueber.
  2. Lassen Sie es in nativer Aufloesung laufen. Stellen Sie das OS-Display auf die angegebene Aufloesung des Monitors ein (oft QHD 1440p oder 4K UHD). Herunterskalieren erweicht Defekte; nativ laufen zeigt jeden Pixel im Verhaeltnis 1:1.
  3. Zyklisieren Sie alle sechs Farben im Vollbild. Bewegen Sie fuer jede Farbe Ihre Augen langsam ueber das Panel - oben-links, oben-rechts, Mitte, unten-rechts, unten-links. Verbringen Sie 20-30 Sekunden pro Farbe. Beachten Sie jeden Pixel, der nicht zum umgebenden Feld passt.
  4. Fotografieren Sie jeden Defekt. Verwenden Sie eine Telefonkamera mit ausgeschaltetem Blitz, fokussiert auf den Bildschirm. Machen Sie eine Weitwinkelaufnahme, die den gesamten Bildschirm zeigt, und eine Nahaufnahme jedes Defekts. Schliessen Sie das Farbfeld ein, das den Defekt zeigt (toter Pixel zeigt sich am besten auf Weiss; festsitzender Pixel zeigt sich am besten auf der gegenueberliegenden Farbe).
  5. Deaktivieren Sie Night Shift, Blaulichtfilter und HDR. Diese Funktionen verschieben Farben; die Defekterkennung benoetigt eine neutrale Ausgabe.

Ueber tote Pixel hinaus - andere haeufige Fehler bei neuen Monitoren

Hintergrundbeleuchtungsblutung. Suchen Sie auf einem soliden schwarzen Feld nach unebenmaessigem Glanz nahe der Bildschirmkanten. Etwas Blutung ist auf IPS-Panels unvermeidlich; starke Blutung (helle Ecken, ungleichmaessige Streifen) ist ein Defekt.

Farbgleichmaessigkeit. Suchen Sie auf einem soliden mittelgrau (50% grau) nach rosa/gruenen/gelben Toenen in verschiedenen Regionen. Ideale Gleichmaessigkeit ist ein perfekt einheitliches Graufeld; subtile Variation ist normal; starke regionale Toene weisen auf einen Paneldefekt hin.

Banding. Suchen Sie auf einem glatten Graustufengradient (unser LCD-Test kann einen rendern) nach sichtbaren Stufen zwischen Farbbaendern. 8-Bit-Panels haben etwas Banding; 10-Bit-Panels sollten glatt sein.

Tote Reihen oder Spalten. Selten aber schwerwiegend. Eine ganze horizontale oder vertikale Linie, die tot oder festgesetzt ist. Eindeutig im Farbzyklus. Grund fuer sofortige Rueckgabe.


Was Hersteller und Einzelhaendler akzeptieren

Akzeptanzrichtlinien variieren je nach Anbieter und Region. Uebliche Schwellen (pruefen Sie Ihren spezifischen Anbieter):

Dell, HP, LG, Samsung Pro-Lines. Typischerweise Null-Tote-Pixel-Garantie auf Premium-Monitoren; jeder tote Pixel innerhalb von 30 Tagen ist eine Rueckgabe. Budget-Konsumenten-Monitore benoetigen oft 3 oder mehr tote Pixel innerhalb einer angegebenen Region, bevor eine Rueckgabe akzeptiert wird.

Apple Studio Display, Pro Display XDR. Jeder defekte Pixel ist Grund fuer einen Umtausch innerhalb des 14-Tage-Rueckgabefensters.

Amazon, Best Buy, Big-Box-Einzelhaendler. Ihre Richtlinien sind in der Regel grosszuegiger als die Herstellergarantie - ein toter Pixeldefekt innerhalb des Rueckgabefensters des Einzelhaendlers (oft 30 Tage) wird mit Nachweis akzeptiert.


Dokumentation, die hilft, dass eine Rueckgabe durchgeht

Wenn die Rueckgabe angefochten wird oder zur Garantiepruefung geht, spart Dokumentation Zeit:

  • Telefonfoto jedes Defekts. Nahaufnahme mit dem defekt-aufdeckenden Farbfeld sichtbar.
  • Zeitgestempelter Screenshot des LCD-Tests im Vollbild laufend (die meisten Einzelhaendler akzeptieren das als Beweis, dass der Test durchgefuehrt wurde).
  • Seriennummer des Monitors (normalerweise auf einem Aufkleber auf der Rueckseite).
  • Kaufbeleg mit dem Kaufdatum.

Wenn Sie einen echten Defekt finden: Rueckgabe-Weg

Kontaktieren Sie zuerst den Einzelhaendler; sein Rueckgabeprozess ist normalerweise einfacher als die Herstellergarantie. Die meisten Einzelhaendler bearbeiten eine Rueckgabe innerhalb von 48 Stunden, wenn der Defekt im Testnachweis verifiziert ist. Wenn das Rueckgabefenster des Einzelhaendlers abgelaufen ist, Sie aber noch innerhalb der Herstellergarantie sind (typischerweise 12-36 Monate), eskalieren Sie an den Hersteller mit demselben Nachweis-Set.


Wenn Sie einen festsitzenden Pixel finden: versuchen Sie zuerst zu loesen

Bevor Sie akzeptieren, dass ein festsitzender Pixel permanent ist, versuchen Sie die "Pixel-Massage"-Technik: ein feuchtes Mikrofasertuch, sanfter Druck direkt auf den festsitzenden Pixel fuer 5-10 Sekunden, einige Male wiederholen. Alternativ lassen Sie ein Pixel-Zyklus-Muster fuer 6-24 Stunden laufen (schnelles Farbflackern an der festsitzenden Pixelstelle taut oft den Fluessigkristall auf). Erfolgsquote ist vielleicht 30-40% - es lohnt sich vor dem Einleiten einer Rueckgabe zu versuchen, wenn Sie ueber das einfache Rueckgabefenster hinaus sind.
