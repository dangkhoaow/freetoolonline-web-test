Was ein Zip-Dateigroesse-Komprimierer wirklich macht

Der Komprimierer nimmt einen Ordner mit Dateien, fuehrt ZIP-Kompression auf jedem Eintrag aus und schreibt sie in ein einziges .zip-Archiv. Die resultierende Datei ist kleiner als die Summe der Originaldateien bei den meisten Eingaben - die Einsparungen kommen vom Entfernen von Redundanz innerhalb von textfoermigem Inhalt (Logs, Quellcode, CSVs, JSON). Bei Eingaben, die bereits komprimiert sind (JPG-Fotos, MP4-Videos, moderne Office-Dokumente), spart die .zip-Huelle Bytes nur auf Verzeichnisstruktur-Ebene, nicht innerhalb des Dateirumpfs. Erwarten Sie moderate Groessenreduktionen bei Fotoordnern und grosse Reduktionen bei Textordnern.

Wie man einen Ordner in eine einzige .zip komprimiert

  1. Oeffnen Sie Zip File in diesem Browser-Tab.
  2. Ziehen Sie den Ordner in die Upload-Zone (oder waehlen Sie ihn ueber den Datei-Auswaehler).
  3. Optional: geben Sie ein Passwort in das Passwort-Feld ein, wenn Sie das Archiv gesperrt haben moechten.
  4. Klicken Sie auf Convert. Das Archiv wird auf dem freetoolonline-Service-Host gebuendelt.
  5. Laden Sie die .zip zurueck auf Ihr Geraet. Die Originale auf Ihrer Maschine bleiben unberuehrt.

Wann diese Anleitung die richtige Wahl ist

Diese Anleitung beantwortet die Aufgabe "ich habe einen Ordner, ich will eine kleinere .zip". Wenn die Eingabe ein einzelnes Bild statt eines Ordners ist, wechseln Sie zu Compress Image. Wenn die Eingabe ein PDF ist, wechseln Sie zu Compress PDF. Wenn Sie bereits eine .zip haben und sie kleiner machen wollen, ist das richtige Werkzeug Wie man eine Zip-Datei kleiner macht - das deckt erneutes Zippen mit entfernten Medien und das Aufteilen einer grossen .zip auf Volumes ab.

Haeufig gestellte Fragen

Ist ein Zip-Dateigroesse-Komprimierer dasselbe wie das Komprimieren jeder Datei innerhalb der zip?

Das Zip-File-Werkzeug fuehrt ZIP-Kompression auf jeder Datei aus, waehrend es sie buendelt. Bei Dateien, die bereits in einem komprimierten Format vorliegen (JPG, MP4, moderner Office), kommt der Grossteil der Einsparungen aus der Verzeichnishuelle, nicht aus dem Dateirumpf. Bei textfoermigen Dateien (CSV, JSON, Quellcode, Logs) ist die Pro-Datei-Kompression selbst die Haupteinsparung.

Haelt das ZIP-Passwort den Inhalt privat, wenn jemand den Download abfaengt?

Das ZIP-Passwort fuegt eine Schicht hinzu, die beilaeufiges Entpacken ohne Passwort blockiert. Es ist keine Ende-zu-Ende-Verschluesselung - der freetoolonline-Service-Host sieht den Dateiinhalt waehrend des Buendelns, und der Download laeuft ueber HTTPS. Bei hoch sensiblen Inhalten (Kontoauszuege, medizinische Unterlagen) verschluesseln Sie lokal, bevor Sie hochladen.

Wie lange haelt der Download-Link?

Die Download-URL zeigt auf eine Kopie, die auf dem freetoolonline-Service-Host fuer ein kurzes Rotationsfenster gehalten wird. Nach der Rotation gibt dieselbe URL 404 zurueck, selbst wenn Sie sie als Lesezeichen gespeichert haben. Fuehren Sie die Kompression erneut aus, um eine frische URL zu erhalten.
