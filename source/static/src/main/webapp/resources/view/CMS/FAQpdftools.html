Frequently Asked Questions

Which tool should I pick when I need one PDF from a set of images?

Use Images to PDF. It preserves the order you upload the files and produces a single multi-page PDF. If you need to mix images, existing PDFs, and blank pages, pick Compose PDF instead.

My PDF-to-text extraction is blank - why?

Your PDF is a scanned image of text, not a digital text layer. PDF to Text extracts the text layer directly; scanned PDFs need OCR first. If you only have the scan, use a dedicated OCR tool before converting.

Are my PDFs uploaded to your server?

The hub catalog page itself never receives your files - it is a discovery surface that lists the PDF tools and routes you to the one your task belongs on. Once you click a member tool, that tool runs through a short-lived HTTPS processing service: the file is uploaded over a secure connection, the requested operation (split, protect, preflight, flatten, PDF-to-text, PDF-to-HTML, compose, join, images-to-PDF, remove-password) runs server-side, and the result is streamed back to your device for download.

The working copy is released as soon as the response is delivered; the file is not retained on a permanent server, no account or sign-in is required, and the file is not indexed or re-used. Lightweight read-only inspections (file size, page count, format check before you press the action button) run locally in the browser. Every transformation that produces a new file or a downloadable report runs on the processing service - not in the browser. If a particular document is too sensitive to leave the device for any reason, pick a desktop PDF tool for that one file; for everything else, the service is the trade-off for getting the work done in seconds without an install.

Two practical signals tell you the tool you opened is using the service: the upload button accepts files larger than a few megabytes, and the tool produces a downloadable output file (a new PDF, a text export, an image export, or a pass/fail report). When both signals are present, the file is going through the processing service.

Which PDF tool reads a file vs writes a file?

Read-only (no new PDF written): Preflight PDF reports pass/fail against PDF/A profiles, PDF to Text extracts a `.txt`, PDF to Images emits one image per page, PDF to HTML emits an HTML export. Write (new PDF produced): Compose PDF, Split PDF by Range, Join PDFs, Protect PDF, Remove PDF Password, Flatten PDF, Images to PDF. The split between the two columns is what determines whether the tool needs an existing PDF as input or accepts other formats (or no input).

Can I password-protect a PDF for sensitive documents?

Yes - use PDF Password or Protect PDF to add or manage a password. Password protection is useful for email delivery; for highly sensitive documents, pair it with a secure sharing channel and not just email.

What if I need to convert a PDF to images instead of text?

Use PDF to Images. Each page becomes a separate JPG or PNG you can embed, edit, or share. For a single HTML export of the whole PDF, pick PDF to HTML.
