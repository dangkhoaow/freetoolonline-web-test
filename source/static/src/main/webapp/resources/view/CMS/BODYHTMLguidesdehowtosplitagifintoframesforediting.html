Wie Man Ein GIF In Einzelbilder Aufteilt

Zuletzt geprueft 2026-04-27. Oeffnen Sie Extract GIF to Frames, um in Ihrem Browser ohne Hochladen des GIF aufzuteilen.

30-Sekunden-Antwort. Ziehen Sie das GIF auf Extract GIF to Frames; das Tool dekodiert jedes Einzelbild und bietet einen Download fuer jedes an. Frame-Nummerierung und Per-Frame-Timing werden mit uebertragen, sodass Sie die Animation spaeter neu erstellen koennen, falls noetig. Die Arbeit erfolgt in Ihrem Browser; nichts wird hochgeladen.

Der Fuenf-Schritte-Workflow

  1. Oeffnen Sie das Tool. Extract GIF to Frames funktioniert in jedem modernen Browser. Kein Login, keine App-Installation.
  2. Ziehen Sie das GIF. Ziehen Sie es von Ihrem Desktop oder waehlen Sie aus einer Telefon-Bibliothek. Der Decoder liest jedes Einzelbild der Reihe nach.
  3. Bestaetigen Sie die Frame-Anzahl. Das Tool meldet die Gesamtzahl der gefundenen Frames und die Per-Frame-Verzoegerung. Vergleichen Sie mit dem, was Sie erwarten - ein 3-Sekunden-15-fps-GIF sollte 45 Frames erzeugen.
  4. Waehlen Sie das Ausgabeformat. PNG bewahrt Transparenz und ist verlustfrei; die richtige Wahl fuer die Bearbeitung. JPG ist kleiner; richtig, wenn die Frames Fotografien sind und Speicher wichtiger ist als Praezision.
  5. Herunterladen. Per-Frame-Downloads streamen in Ihren Downloads-Ordner, benannt mit Sequenznummern (frame-001.png, frame-002.png, ...).

Was Ihnen das Per-Frame-Timing sagt

GIF-Frames tragen ihren eigenen Verzoegerungswert - die Millisekunden, die der Frame vor dem naechsten angezeigt werden soll. Die meisten GIFs verwenden eine einheitliche Verzoegerung (40 ms = 25 fps, 100 ms = 10 fps). Einige Animationen verwenden variable Verzoegerungen - eine lange Pause bei Schluesselframes, schnelle Bewegung dazwischen. Variable Verzoegerungen ueberleben die Aufteilung, sodass das Wieder-Enkodieren zurueck zu GIF oder zu MP4 das urspruengliche Timing reproduziert.

Wenn die Timing-Daten kritisch sind (Sie debuggen eine Animation, archivieren mit Absicht zur Wiederherstellung), behalten Sie die Per-Frame-Verzoegerungsmetadaten, die das Tool zusammen mit den Bildern exportiert. Die meiste Bearbeitungssoftware liest sie zurueck, wenn Sie wieder zusammenbauen.

Warum ueberhaupt aufteilen

  • Ein einzelnes Einzelbild bearbeiten. Korrigieren Sie ein Frame, fuegen Sie eine korrigierte Version ein, kodieren Sie neu. Schneller als das Neuaufnehmen eines Bildschirmfotos.
  • Standbilder fuer ein Thumbnail oder Vorschau ziehen. Die meisten Social-Plattformen spielen GIFs automatisch ab, aber ein statisches PNG-Thumbnail ist das, was Indexierer und Link-Vorschauen verwenden.
  • GIF in ein effizienteres Format konvertieren. Moderne Formate - WebP, AVIF, MP4 ohne Audio - alle beginnen mit Frames-plus-Timing. Aufteilen ist der erste Schritt.
  • Archivieren. Ein Ordner mit PNG-Frames ist eine stabile Darstellung. GIF, wie jedes Multi-Frame-Format, ist ein Wrapper um die gleichen zugrunde liegenden Frame-Daten.

In die andere Richtung - ein GIF aus Frames bauen

Das Gegenteil - Frames zurueck zu einer Animation zusammenzubauen - ist das, was GIF Maker tut. Ziehen Sie einen Ordner mit sequenziell nummerierten Bildern, setzen Sie die Frame-Verzoegerung, und das Tool schreibt ein frisches GIF. Verwenden Sie dieselbe Verzoegerung, mit der Sie begonnen haben, um das urspruengliche Timing zu treffen, oder setzen Sie einen anderen Wert, um die Animation zu beschleunigen oder zu verlangsamen.
