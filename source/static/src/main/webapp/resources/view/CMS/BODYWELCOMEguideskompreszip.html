Sebelum membuka pembuat arsip, susun lebih dulu masukan menurut jenis di folder lokal Anda. Berkas teks (catatan, ekspor CSV, file sumber, dump JSON) ke satu bundle; media yang sudah dipadatkan (foto JPG, video MP4, audio MP3) ke bundle lain. Bundle teks adalah tempat wadah ZIP benar-benar memangkas byte; bundle media tetap melewati arsip yang sama, tetapi sebagian besar untuk kerapihan pengiriman, bukan untuk pengurangan ukuran.
