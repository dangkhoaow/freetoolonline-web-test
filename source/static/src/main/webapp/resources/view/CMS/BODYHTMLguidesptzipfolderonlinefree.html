Zipar Pasta Online Gratis

Precisa transformar uma pasta inteira em um unico .zip sem instalar nada? Solte a pasta na ferramenta do navegador, os conteudos viajam via HTTPS para o servico de processamento, e o arquivo empacotado volta como um link de download. Sem conta, sem instalacao, gratis para usar.

Ferramenta de implementacao/zip-file.html

Os tres passos

  1. Abra o zipador de pasta em /zip-file.html. Arraste a pasta para o slot de upload.
  2. Opcional: defina uma senha via zip com senha.
  3. Baixe o .zip. Quando a barra de progresso terminar, o link aponta para o arquivo empacotado.
