Frequently Asked Questions

Is this an actual browser extension I need to install?

No - this page is a web tool that gives readers the same paste-and-format experience an extension would. Open the page, paste the JSON, copy the formatted output. There is nothing to install.

How is this different from the main JSON formatter page?

The transform logic is identical. This page is framed for readers who searched "JSON formatter extension" expecting an install-style workflow; the lead copy explains the in-tab format flow they actually want. If you want the canonical entry point, use /json-formatter.html; if you want the read-only viewer, the editor, or the diff, switch to the matching sibling page.

Does the JSON get uploaded anywhere?

No - the formatting runs locally in your browser. Nothing is sent to a server. Confidential payloads can be formatted without leaving the tab.

How do I change the indentation width?

The indent width selector sits above the input pane. Pick 2-space, 4-space, or tab; the right pane re-renders the moment you change the option.
