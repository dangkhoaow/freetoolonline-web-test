Como Comprimir una Carpeta a Zip Online, Paso a Paso

Ultima revision 2026-04-27. Abre la herramienta de zip para comprimir una carpeta en tu navegador sin subirla.

Respuesta en 30 segundos. En el escritorio, arrastra la carpeta a la herramienta de zip - el navegador recorre el arbol de directorios y lo preserva dentro del archivo. En el movil, los navegadores no pueden leer carpetas directamente, asi que la solucion alternativa es pulsar y mantener la carpeta, elegir "comprimir" (integrado en iOS y Android moderno), o arrastrar la carpeta a traves de un gestor de archivos. La ruta de escritorio es mas rapida y funciona en todos los navegadores modernos.

Cinco pasos en el escritorio

  1. Elige la carpeta. Arrastra la carpeta a la zona de soltar en la herramienta de zip. El navegador usa la API File System Access o un fallback de selector de directorio y lee cada archivo dentro, incluidas las subcarpetas anidadas.
  2. Manten la estructura. La herramienta, por defecto, preserva la disposicion de la carpeta dentro del zip. Los destinatarios descomprimen y ven my-folder/sub/file.txt exactamente como lo enviaste. No aplanes a menos que necesites especificamente una disposicion plana - las herramientas de descompresion manejan estructuras anidadas de forma nativa, y los metadatos hacen que el archivo sea autodocumentado.
  3. Ajusta la compresion. El predeterminado (nivel 6) es adecuado para contenido mixto. El mejor (nivel 9) ayuda para carpetas dominadas por documentos, codigo fuente o registros. Almacenar (nivel 0) es la opcion correcta para carpetas de fotos, videos o audio - todo dentro ya esta comprimido.
  4. Nombra la salida. Usa un nombre de archivo descriptivo: project-handover-2026-04.zip es mejor que archive.zip. Evita espacios y caracteres especiales al enviar por correo - algunos clientes aun recodifican el nombre del archivo.
  5. Descarga. El zip se construye localmente. La carpeta original queda intacta.

La solucion alternativa para movil

Los navegadores moviles no pueden dar a un sitio web acceso directo a una carpeta - solo exponen archivos individuales. Tres rutas que funcionan:

  • App Archivos de iOS. Pulsa y manten la carpeta, elige Comprimir. La salida es un zip en la misma ubicacion. Abrelo en cualquier app o compartelo.
  • Android. La mayoria de las apps de gestion de archivos (Files by Google, Solid Explorer, MiXplorer) incluyen una accion Comprimir con la misma forma - pulsa y manten, Comprimir.
  • Multi-seleccion del lado del navegador. Selecciona todos los archivos dentro de la carpeta (uno a la vez en iOS Safari, multi-seleccion en la mayoria de navegadores Android) y sueltalos en la herramienta de zip. Pierdes la estructura anidada a menos que la recrees manualmente.

Trampas de tamano a evitar

Una carpeta con un solo archivo gigante dentro (un video, un volcado de base de datos) se comprime muy poco porque ese archivo es el cuello de botella. Comprimelo o recortalo primero. Una carpeta con 20.000 archivos diminutos se comprime mucho pero crea un archivo lento - algunas herramientas de descompresion expiran en archivos con demasiadas entradas. Si ves ese patron, agrupa los archivos pequenos en subcarpetas o pre-empaquetalos en un tar antes de comprimir.

Verifica el zip antes de enviar

Abre el zip descargado localmente para confirmar la estructura y que cada archivo esperado esta dentro. Descomprimir un archivo en el navegador listara el contenido sin extraer. Para los trade-offs del formato, ver online zip vs 7z vs rar o el hub de herramientas zip.
