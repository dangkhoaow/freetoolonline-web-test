A GIF is one file that plays as motion; the individual frames inside it are the still images that make up that motion. When the goal is to grab one frame for a thumbnail, separate every frame for a storyboard, or rebuild the sequence in a different editor, the extractor reads the animation, splits it on the timeline, and writes each frame to a separate PNG or JPG file. The animation stays intact; the output is a new set of stills sitting alongside the source.
