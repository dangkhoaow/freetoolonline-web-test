Implementing tool/zip-file.html

The implementing tool sits at /zip-file.html and accepts both individual files and folders. Pick the items you want bundled, decide whether to add a password (optional, and removable later via Remove ZIP password), then click the convert button. The output is a single .zip archive your browser downloads to its default folder. For a more granular look at what the converter does and does not do (compression ratios, included metadata, supported source file types), see the companion disambiguation guide Zip File Converter - what it actually does.
