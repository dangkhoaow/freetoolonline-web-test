Comprimir Archivo Zip

Comprimir archivo zip es la forma de empaquetar un archivo, varios archivos o una carpeta entera en un solo archivo .zip en el navegador. Subes los archivos, el servicio los agrupa en el servidor via HTTPS, los archivos fuente se eliminan automaticamente despues de una breve ventana, y el archivo regresa como una sola descarga.

Lo que realmente hace la herramienta de compresion zip

La herramienta en /zip-file.html realiza tres trabajos principales en una pagina. Primero, agrupa: un archivo, un conjunto de archivos o una carpeta entera (las subcarpetas se preservan) se convierte en un solo archivo .zip que el destinatario descarga con un clic. Segundo, nombra el archivo a traves del panel Configuracion para que el destinatario vea el tema o el nombre del proyecto en el nombre del archivo en lugar de una marca de tiempo generica. Tercero, puede bloquear el archivo con una contrasena opcional: elige el metodo de cifrado Estandar, AES-128 o AES-256 cuando la contrasena se envie por un canal separado del archivo. Todo el procesamiento ocurre en nuestros servidores via conexion HTTPS, y los archivos fuente se eliminan automaticamente despues de una breve ventana de retencion.

Comprimir archivo zip realmente reduce el tamano

El archivo de salida de /zip-file.html es mas pequeno que la suma de sus entradas solo cuando los archivos que uniste pueden ser comprimidos por el algoritmo deflate ZIP. Si agrupas archivos de texto largos, logs diarios, exportaciones CSV, codigo fuente o documentos Office, la secuencia de bytes repetida dentro de ellos permite que ZIP comprima la salida al rango de 30 a 60 por ciento del tamano de origen. Por el contrario, las entradas que ya son medios comprimidos - JPG de camara digital, clips de video MP4, canciones MP3 o adjuntos PDF cuyo contenido es imagen - ya fueron comprimidos por sus respectivos codecs antes de llegar aqui, asi que ZIP solo los envuelve: el tamano final se mueve alrededor de 0 a 5 por ciento y a veces aumenta ligeramente debido a la sobrecarga de los nombres de archivo y la tabla del archivo. El resumen sigue siendo uno: ZIP comprime lo que aun no esta comprimido; ZIP envuelve lo que ya esta comprimido. Si tu objetivo es cumplir con el limite de 25 MB de adjunto de correo electronico u otro limite de subida, primero reduce los medios pesados en la herramienta de compresion de imagen o video, luego combina los resultados en un solo archivo ZIP en esta pagina.

Privacidad y manejo de datos

Esta pagina se ejecuta sin cuenta: no necesitas registrarte, iniciar sesion o instalar una extension del navegador antes de agrupar el archivo. Los archivos de entrada se envian via HTTPS a nuestros servidores de procesamiento, el archivo .zip se construye alli, y los archivos fuente se eliminan automaticamente despues de una breve ventana de retencion para que no se acumulen en el lado del servidor. El enlace de descarga generado apunta solo al archivo final - no a una copia de tus entradas. Para archivos que pasaran por un canal de compartir de baja confianza, activa la contrasena y elige el metodo AES-128 o AES-256 en el panel Configuracion, luego envia la contrasena por un canal diferente al del archivo para que un lector que intercepte un lado no abra automaticamente el otro.

Como comprimir archivo zip en el navegador, paso a paso

  1. Abre /zip-file.html en una pestana nueva.
  2. Arrastra o selecciona los archivos que quieres agrupar. Una carpeta tambien se acepta; las subcarpetas se preservan dentro del archivo.
  3. Opcional: abre Configuracion, nombra el archivo y agrega una contrasena mas el metodo de cifrado (Estandar, AES-128 o AES-256) si el destinatario ya tiene un canal separado para la contrasena.
  4. Haz clic en el boton Zip. La pagina subira las entradas via HTTPS, construira el archivo en el servidor y mostrara el enlace de descarga.
  5. Guarda el archivo .zip resultante. Si estableciste una contrasena, enviala por una ruta diferente al archivo.

Preguntas que aparecen con frecuencia

Necesito registrarme o instalar una aplicacion?

No. Esta pagina se ejecuta en el navegador y no pide cuenta. Solo arrastras los archivos, presionas Zip y descargas el resultado.

Mis archivos estan seguros?

Las subidas se envian via HTTPS al servidor de procesamiento, el archivo se construye en el lado del servidor, y los archivos fuente se eliminan automaticamente despues de una breve ventana de retencion. Para una capa adicional, usa contrasena con el metodo AES-128 o AES-256 y envia la contrasena por un canal separado.

Por que el tamano de mi archivo no se redujo mucho?

Probablemente tus entradas contienen medios que ya fueron comprimidos por sus respectivos codecs (JPG, MP4, MP3 o PDF con imagenes). ZIP solo envuelve esos archivos sin reducirlos mucho. Primero comprime los medios pesados con una herramienta de imagen o video, luego agrupa los resultados.
