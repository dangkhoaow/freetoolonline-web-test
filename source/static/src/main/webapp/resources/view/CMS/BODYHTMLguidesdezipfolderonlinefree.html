Ordner Online Kostenlos Zippen

Muessen Sie einen ganzen Ordner in eine einzige .zip umwandeln, ohne etwas zu installieren? Legen Sie den Ordner in das Browser-Werkzeug, der Inhalt wird ueber HTTPS an den Verarbeitungsdienst gesendet, und das gepackte Archiv kommt als Download-Link zurueck. Kein Konto, keine Installation, kostenlos zu verwenden.

Die drei Schritte

  1. Oeffnen Sie den Ordner-Zipper unter /zip-file.html. Ziehen Sie den Ordner in den Upload-Slot.
  2. Optional: setzen Sie ein Passwort ueber Zip mit Passwort.
  3. Laden Sie die .zip herunter. Wenn der Fortschrittsbalken fertig ist, zeigt der Link auf das gepackte Archiv.
