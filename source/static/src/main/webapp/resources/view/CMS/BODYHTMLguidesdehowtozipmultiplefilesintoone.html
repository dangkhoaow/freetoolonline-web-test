Wie man mehrere Dateien zu einer ZIP zusammenfuegt

Zuletzt geprueft: 2026-04-27. Oeffnen Sie das ZIP-Werkzeug, um Dateien in Ihrem Browser ohne Upload zu kombinieren.

30-Sekunden-Antwort. Mehrere Dateien zu einer ZIP zu buendeln erfordert vier Schritte in dem ZIP-Werkzeug: Dateien auswaehlen, Ausgabe benennen (etwas Beschreibendes, ohne Leerzeichen wenn E-Mail das Ziel ist), Default-Kompression fuer gemischten Inhalt oder Store fuer bereits komprimierte Medien einstellen und auf Download klicken.

Die vier Schritte im Detail

  1. Dateien auswaehlen. Drag-and-Drop ist der schnellste Weg auf dem Desktop; Multi-File-Picker funktioniert auf Telefonen. Das Werkzeug akzeptiert jeden Dateityp.
  2. Ausgabe benennen. Das Werkzeug nimmt standardmaessig archive.zip - funktioniert, aber sagt dem Empfaenger nichts. 2026-Q1-rechnungen.zip oder urlaubsfotos.zip lesen sich besser.
  3. Kompressionslevel einstellen. Default (Level 6) fuer einen Mix. Best (Level 9) wenn die Dateien hauptsaechlich Text sind. Store (Level 0) wenn alles darin bereits komprimiert ist.
  4. Herunterladen. Der Browser uebergibt Ihnen das fertige Archiv. Die Originaldateien bleiben unberuehrt.

Was das Werkzeug nicht tut

  • Ein Passwort waehrend der Erstellung hinzufuegen. Fuer passwortgeschuetzte Archive siehe ZIP-Datei mit Passwort.
  • Ordnerstruktur ueber mehrere Drag-and-Drop-Auswahlen erhalten. Wenn Sie ein verschachteltes Layout in der ZIP benoetigen, ziehen Sie den uebergeordneten Ordner.
  • Eine Datei vor dem Zippen bearbeiten. Komprimieren Sie Bilder zuerst mit Bild komprimieren wenn die Groesse wichtig ist.
