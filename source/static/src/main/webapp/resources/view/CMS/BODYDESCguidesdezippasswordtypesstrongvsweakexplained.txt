Bevor Sie eine passwortgeschuetzte ZIP an /remove-zip-password.html hochladen, pruefen Sie den Verschluesselungstyp. Legacy ZIP 2.0 + ein kurzes oder numerisches Passwort ist in Sekunden wiederherstellbar. WinZip AES-256 mit einem starken Passwort nicht. Ein 30-Sekunden-Entscheidungsleitfaden, damit Sie keinen Upload verschwenden.
