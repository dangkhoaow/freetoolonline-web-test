Was Ein LCD-Test Macht (und Wann Einen Ausfuehren)

Zuletzt geprueft 2026-04-27. Der Bildschirmtest fuehrt Vollbild-Farbsweeps in Ihrem Browser aus - kein Installer, kein Treiber, kein Upload. Funktioniert auf jedem Desktop, Laptop, Tablet und Telefon mit modernem Browser.

30-Sekunden-Antwort. Ein LCD-Test fuellt Ihren Bildschirm mit Volltonfarben (Rot, Gruen, Blau, Weiss, Schwarz, manchmal Grauschritte), damit Defekte des Panels gegen einen einheitlichen Hintergrund sichtbar werden. Die vier Dinge, die er zuverlaessig erkennt: tote Pixel (immer schwarze Punkte), festgesetzte Pixel (immer eine Farbe, unabhaengig vom Input), Lichthof (ungleichmaessige Helligkeit nahe der Raender bei schwarzem Bildschirm) und Banding / Dithering-Artefakte (sichtbare Stufen in einem Verlauf). Fuehren Sie ihn aus, bevor das Rueckgabefenster eines neuen Monitors schliesst, bevor Sie einen generalueberholten Bildschirm akzeptieren, oder wann immer Sie einen "komischen Fleck" auf Ihrem Display vermuten. Oeffnen Sie das LCD-Test-Werkzeug und gehen Sie die Farben durch.

Was "LCD-Test" wirklich bedeutet

Der Begriff "LCD-Test" ist ein Sammelname fuer eine kleine Familie von Tests, die unterschiedliche Defektklassen pruefen. Die browserbasierte Version - die diese Anleitung behandelt - ist die haeufigste, weil sie nichts kostet, ueberall laeuft und die Defekte erkennt, die einem Kaeufer am wichtigsten sind. Sie ist nicht die einzige Option (Hersteller fahren in der Fabrik aufwendigere Kalibrierungs-Suites), aber sie ist das, was 99% der Menschen, die "lcd-test" bei Google eingeben, tatsaechlich wollen.

Der Mechanismus ist einfach: das Werkzeug fuellt den gesamten Bildschirm mit einer Volltonfarbe. Ein perfektes Panel ist ein perfekt einheitliches Feld. Defekte zeigen sich als Abweichungen - ein winziger Punkt anderer Farbe, ein hellerer oder dunklerer Fleck nahe der Ecke, ein Band, das nicht sanft ins naechste uebergeht. Das blosse Auge erkennt Abweichungen von der Gleichmaessigkeit gut, also braucht der Test keine Spezial-Hardware, um nuetzlich zu sein.

Die vier Defekte, die ein LCD-Test erfasst

  • Tote Pixel. Ein Subpixel, das dauerhaft aus ist. In einem roten Feld sieht ein totes Pixel schwarz aus. In einem gruenen Feld ebenfalls schwarz. In einem schwarzen Feld ist es unsichtbar. Der Tot-Pixel-Test geht durch Rot, Gruen, Blau und Weiss, sodass jedes ausgefallene Subpixel auf mindestens einem Hintergrund auftaucht. Leitfaden zum Tot-Pixel-Test behandelt die Diagnosemuster im Detail.
  • Festgesetzte Pixel. Ein Subpixel, das dauerhaft mit voller Intensitaet an ist. Ein festgesetztes rotes Subpixel erscheint als heller roter Punkt auf schwarzem Feld, auf blauem Feld und auf weissem Feld. Tot von festgesetzt zu unterscheiden ist wichtig: festgesetzte Pixel lassen sich manchmal durch ein "Pixel-Exerciser"-Video wiederbeleben, das 10-30 Minuten lang schnell die Farben wechselt; tote Pixel sind verloren.
  • Lichthof. Licht, das an den Raendern eines LCD-Panels austritt, wo das Backlight nicht perfekt eingeschlossen ist. Am sichtbarsten auf schwarzem Feld in dunklem Raum - Sie sehen ungleichmaessiges Leuchten nahe der Ecken oder entlang einer Kante. Eine kleine Menge ist auf jedem IPS-Panel normal; viel ist ein Defekt. Die meisten Garantien decken "uebermaessig" ab, vom Hersteller mehrdeutig definiert.
  • Banding und Dithering-Artefakte. Ein hochwertiges Panel kann sanfte Verlaeufe ohne sichtbare "Stufen" rendern. Ein Panel mit begrenzter Farbtiefe (6-Bit statt 8-Bit oder 8-Bit statt 10-Bit) produziert sichtbare Baender, wenn der Verlauf einen weiten Wertebereich abdeckt. Das Graustufen-Schrittmuster im LCD-Test deckt das auf.

Wann das Ausfuehren eines LCD-Tests die richtige Entscheidung ist

Fuenf Situationen, in denen der Test die zwei Minuten wert ist:

  • Bevor das Rueckgabefenster eines neuen Monitors oder Laptops schliesst. Die meisten Monitore haben eine 14-30-Tage-Rueckgaberichtlinie plus Herstellergarantie. Die Rueckgaberichtlinie erlaubt das Umtauschen einer defekten Einheit ohne Nachfragen; die Garantie erfordert oft, dass Sie den Defekt in einem Videoanruf mit dem Support nachweisen. Fuehren Sie den Test in der ersten Woche aus, solange Sie es noch einfach zurueckgeben koennen.
  • Bevor Sie einen generalueberholten oder B-Ware-Bildschirm akzeptieren. Refurbished-Preise sind attraktiv, aber die Einheit kann ererbte Defekte haben, deretwegen der Vorbesitzer sie zurueckgab. Fuehren Sie den Test am Tag 1 aus.
  • Bevor Sie einen Telefon- oder Tablet-Trade-in behalten. Carrier und Wiederverkaeufer versenden gelegentlich Trade-in-Geraete mit verstecktem Bildschirmschaden. Der Pixel-Test bringt es innerhalb des Rueckgabefensters des Carriers zum Vorschein.
  • Wenn Sie einen "komischen Fleck" auf einem bereits vorhandenen Display vermuten. Ein Staubkorn auf dem Bildschirm sieht aus einer Armlaenge identisch zu einem toten Pixel aus. Der LCD-Test bestaetigt, ob es auf der Oberflaeche liegt (mit Mikrofasertuch abwischen) oder im Panel (Garantiefall).
  • Bevor Sie mit einem Streaming-, Gaming- oder Design-Rig oeffentlich auftreten. Ein Defekt, der auf Ihrem normalen Desktop unsichtbar ist, kann gegen einen Green-Screen-Hintergrund oder eine einheitliche Farbe in einem Videoanruf grell auffallen. Testen Sie das Rig mit den Farben, die Sie tatsaechlich uebertragen.

Und fuer die taegliche Nutzung? Wahrscheinlich auslassen - der Test ist fuer "ist das Panel akzeptabel?", nicht fuer "ist das Panel perfekt fuer Color-Grading kalibriert?". Farbkalibrierung ist ein separater Workflow mit separater Hardware (einem Kolorimeter).

So lesen Sie die Testergebnisse

Gehen Sie jede Farbe vollbild 5-10 Sekunden durch, Augen etwa 30-50 cm vom Panel entfernt. Worauf Sie achten:

  • Pixel-Gleichmaessigkeit. Jeder Punkt, der vom Feld abweicht, ist ein Defektkandidat. Notieren Sie seine Position (z.B. "oben links, etwa 3 cm nach innen"). Wiederholen Sie es ueber alle Farben - ein totes Pixel zeigt sich auf einer Farbe, ein festgesetztes auf drei oder vier.
  • Rand-Gleichmaessigkeit. Auf dem schwarzen Bildschirm im dunklen Raum die vier Raender absuchen. Gleichmaessiges Glow = OK. Helle Ecken oder Streifen = Lichthof.
  • Sanfte Verlaeufe. Gehen Sie die Graurampe langsam durch. Sie sollten einen kontinuierlichen Verlauf von Schwarz zu Weiss sehen, keine sichtbaren Stufen.
  • Farbton-Shift beim Blickwinkel. Neigen Sie den Kopf zur Seite. Ein gutes IPS-Panel haelt Farben bis ~80 Grad ausser-Achse stabil. Ein billiges TN-Panel verschiebt sich schon bei 30 Grad wahrnehmbar. Nuetzlich vor dem Kauf zu wissen.

Ein oder zwei Pixeldefekte auf einem 4K-Panel (8,3 Millionen Pixel) sind statistisch ueblich, und die meisten Garantien decken sie nicht ab. Der Schwellenwert fuer "defekt" variiert: ISO 9241-307 Klasse II erlaubt bis zu 2 tote und 2 festgesetzte Pixel auf einem 4K-Panel. Darueber zurueckgeben.

Warum ein browserbasierter Test genuegt

Sie brauchen keine installierte App, um das zu tun. Der Browser kann bereits Vollbild-Volltonfarben rendern, und das ist der einzige Mechanismus, den der Test braucht. Browserbasierte Tests haben drei weitere Vorteile gegenueber installierten Alternativen:

  • Plattformuebergreifend. Derselbe Test auf Windows, macOS, Linux, ChromeOS, iPadOS, Android, Smart-TV-Browsern. Keine Treiberkompatibilitaet, keine Installationsberechtigung, kein Admin-Passwort.
  • Kein Versions-Drift. Die Seite wird zentral aktualisiert; Sie fahren immer die aktuelle Version. Installierte Werkzeuge altern - letzte Aktualisierung vor drei Jahren ist die Norm.
  • Datenschutz. Keine Datei-Uploads, keine Telemetrie ueber Standard-Webserver-Zugriffslogs hinaus. Der Test ist rein eine Render-Anweisung an Ihren Bildschirm.

Wo browserbasierte Tests zu kurz greifen: Response-Time-, Refresh-Rate- und HDR-Konformitaetstests sind im Browser ohne Spezial-Hardware schwerer. Fuer kompetitives Gaming oder farbkritische Arbeit deckt ein dediziertes Werkzeug (TestUFO, DisplayCAL, Kalibrator-Hardware) diese Luecken ab.

Was der LCD-Test nicht erkennen kann

Nuetzliche Abgrenzungs-Liste:

  • Farbgenauigkeit / Gamut-Abdeckung. Dafuer brauchen Sie ein Kolorimeter.
  • Response-Time / Motion Blur. Dafuer brauchen Sie TestUFO oder eine Hochgeschwindigkeitskamera.
  • Refresh-Rate-Stabilitaet. Dafuer brauchen Sie einen Frame-Time-Logger.
  • Genauigkeit der Touch-Eingabe. Dafuer brauchen Sie ein separates Touch-Test-Werkzeug.
  • Helligkeits-Gleichmaessigkeit in absolutem cd/m^2. Dafuer brauchen Sie ein Luminanz-Messgeraet.

Der LCD-Test erfasst die kategorischen Defekte, die ein Panel disqualifizieren; er misst keine Performance-Qualitaeten. Fuer ein komplettes Pre-Call-Kit siehe Geraete-Test-Checkliste fuer Remote-Arbeit, die Kamera-, Mikrofon- und Tastatur-Tests hinzufuegt.

Den Test ausfuehren

Oeffnen Sie freetoolonline.com/lcd-test.html. Der erste Bildschirm erklaert die Steuerung (Leertaste zum Weiterschalten, Esc zum Verlassen des Vollbilds). Gehen Sie durch Rot, Gruen, Blau, Weiss, Schwarz und die Graurampe. Notieren Sie jeden Defekt, den Sie entdecken. Ist Ihr Panel sauber, sind Sie in unter zwei Minuten fertig. Finden Sie etwas, entscheiden Sie, ob es fuer Rueckgabe qualifiziert - und beginnen Sie das Gespraech, solange das Fenster offen ist.
