Warum "Zip-Datei-Konverter" eine irrefuehrende Suche ist

"In ZIP konvertieren" klingt wie JPG zu PNG oder HEIC zu JPG konvertieren, aber ZIP arbeitet auf einer anderen Schicht. JPG, PNG und HEIC sind Bildformate - sie beschreiben, wie Pixel innerhalb einer Datei codiert sind. ZIP ist ein Archivformat - es wickelt eine beliebige Anzahl von Dateien (jedes Formats) in einen einzigen .zip-Container und wendet allgemeine Komprimierung auf die Bytes an. Die Originaldateien behalten ihre Originalformate innerhalb des Archivs. Wenn Sie entpacken, erhalten Sie die gleichen JPGs, PDFs und DOCXs, mit denen Sie angefangen haben.

Das bedeutet, ein "Zip-Datei-Konverter" macht eigentlich eine von zwei Sachen, je nach Richtung. Hinein: Ihre losen Dateien nehmen und eine einzige .zip produzieren - das ist "erstellen" oder "buendeln", nicht "konvertieren". Hinaus: eine .zip lesen und jede Datei als sich selbst zurueck schreiben - das ist "extrahieren" oder "entpacken", auch nicht "konvertieren". Das Wort "Konverter" bleibt haengen, weil jede andere Dateioperation im Browser Konverter heisst, aber die Mechanik hier ist anders.

Was ZIP wirklich komprimiert (und nicht komprimiert)

ZIP verwendet DEFLATE - einen allgemeinen Entropie-Codierer. Es ist gut darin, Redundanz aus Text, Quellcode, Rohdaten und unkomprimierten Bildern herauszuquetschen. Es macht fast nichts bei Dateitypen, die bereits ihre eigene interne Komprimierung haben. Erwartete Groessenreduzierungen:

  • Text, Code, CSV, Log-Dateien: 60-90% kleiner. Echte Komprimierung.
  • Office-Dokumente (DOCX, XLSX, PPTX): 1-5% kleiner. Diese sind intern bereits ZIP-Dateien.
  • JPG-, PNG-, WebP-, HEIC-Fotos: 0-2% kleiner. Der Bild-Codec hat die Pixel bereits komprimiert.
  • MP3-, AAC-, FLAC-, OGG-Audio: 0-1% kleiner. Bereits entropie-optimiert.
  • MP4-, MOV-, MKV-, WebM-Video: 0-1% kleiner. Am staerksten komprimiertes Format im allgemeinen Gebrauch.

Fuer Ordner, die von Fotos oder Videos dominiert sind, geht es beim ZIP-Schritt hauptsaechlich um Buendeln (ein Download statt fuenfzig), nicht um Schrumpfen. Wenn das Ziel war, eine einzelne grosse Mediendatei tatsaechlich zu schrumpfen, ist ZIP das falsche Tool - greifen Sie zu Bildkomprimierung bei einem Foto (Compress Image), Formatkonvertierung bei iPhone-HEIC (HEIC to JPG), oder einem Datei-Transfer-Link fuer ein Video.

"ZIP zu PDF / JPG / etwas anderem konvertieren"

Diese Formulierung bedeutet normalerweise eine von zwei spezifischen Situationen. Erstens: Sie haben eine .zip erhalten, die eine einzige PDF oder JPG enthaelt, und Sie wollen diese Datei aus dem Archiv. Das richtige Tool ist Unzip File - es liest die .zip und schreibt die Dateien als sich selbst zurueck. Es ist keine Konvertierung beteiligt; die PDF in der .zip ist bereits eine PDF. Zweitens: Sie haben mehrere PDFs, die Sie zusammen packen wollen, und Sie haben aus Versehen "ZIP zu PDF konvertieren" getippt. Das richtige Tool ist Compose PDF - es fuehrt mehrere PDFs zu einem Dokument zusammen. ZIP wuerde sie auch buendeln, aber der Empfaenger muesste zuerst entpacken.

Haeufig gestellte Fragen

Ist ZIP ein Dateiformat, in das ich vor dem Senden "konvertieren" sollte?

Nur wenn Sie mehrere Dateien fuer einen Empfaenger haben. ZIP ist die richtige Antwort, wenn die Alternative ist, fuenf Anhaenge separat zu senden - eine .zip ist einfacher auf beiden Seiten. Fuer eine einzelne Datei ist ZIP unnoetig und schrumpft sie oft nicht. Senden Sie die Datei selbst, oder komprimieren Sie sie direkt mit dem richtigen Tool pro Format (Compress Image fuer Fotos).

Warum endet meine .zip fast in derselben Groesse wie die Originale?

Weil die Originale bereits komprimiert waren. JPG, MP3, MP4 und moderne Office-Dokumente tragen alle ihre eigene Komprimierung intern; DEFLATE innerhalb von ZIP kann die gleichen Bytes nicht ein zweites Mal quetschen. Wenn das Buendel immer noch zu gross zum Senden ist, ist die Antwort kein staerkerer Kompressor - es ist Komprimierung pro Format auf den groessten Elementen oder ein Datei-Transfer-Link fuer Video.

Sollte ich 7z oder RAR statt ZIP verwenden?

Verwenden Sie ZIP, ausser Sie haben einen Grund. ZIP oeffnet nativ auf Windows, macOS, iOS und Android ohne zusaetzliche Software; 7z und RAR brauchen eine separate App auf der Seite des Empfaengers. Der Komprimierungsverhaeltnis-Vorteil von 7z ist klein bei den Dateitypen, die Leute tatsaechlich senden (bereits komprimierte Medien).
