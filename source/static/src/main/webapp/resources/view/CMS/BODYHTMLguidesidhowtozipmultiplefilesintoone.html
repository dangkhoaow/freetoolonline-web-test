Cara Membundel Beberapa File ke Satu ZIP

Direview terakhir: 2026-04-27. Buka alat zip untuk menggabungkan file di browser Anda tanpa mengunggahnya.

Jawaban 30 detik. Membundel beberapa file ke dalam satu zip membutuhkan empat langkah di alat zip: pilih file, beri nama output (sesuatu yang deskriptif, tanpa spasi jika email adalah tujuan), atur kompresi Default untuk konten campuran atau Store untuk media yang sudah terkompresi, dan klik Unduh.

Empat langkah secara rinci

  1. Pilih file. Drag-and-drop adalah jalur tercepat di desktop; pemilih multi-file berfungsi di telepon. Alat menerima semua jenis file.
  2. Beri nama output. Alat secara default archive.zip - berfungsi tetapi tidak memberi tahu apa-apa kepada penerima. 2026-Q1-faktur.zip atau foto-liburan.zip mengurai lebih baik.
  3. Atur level kompresi. Default (level 6) untuk campuran. Best (level 9) ketika file sebagian besar adalah teks. Store (level 0) ketika semua di dalamnya sudah terkompresi.
  4. Unduh. Browser memberikan arsip yang sudah jadi. File asli tetap tidak tersentuh.

Apa yang tidak dilakukan alat

  • Menambahkan kata sandi saat pembuatan. Untuk arsip yang dilindungi kata sandi, lihat zip file dengan kata sandi.
  • Mempertahankan struktur folder lintas beberapa pilihan drag-and-drop. Jika Anda butuh tata letak bersarang di dalam zip, seret folder induk.
  • Mengedit file sebelum zip. Kompres gambar terlebih dahulu dengan kompres gambar jika ukuran penting.

Untuk alur kerja zip lainnya, lihat hub alat zip.
