Kompresor File Zip Online

Seret file, pilihan multi-file, atau seluruh folder ke pembuat ZIP di browser dan alat menulis satu arsip .zip yang berisi semuanya - subfolder dipertahankan, nama file dipertahankan, siap diunduh sebagai satu file. Outputnya adalah yang dimaksud kebanyakan orang ketika mereka berkata "zip ini untuk saya": satu amplop berisi segalanya, lebih kecil dari input ketika input adalah teks berat, kurang lebih sama ukurannya ketika input adalah media yang sudah dikompresi.

Ketika arsip harus bepergian dengan kata sandi

Ketika .zip harus bepergian melalui saluran di mana penerimanya adalah orang asing (tautan bersama, grup chat, email yang diteruskan melalui rantai), pilih metode enkripsi di modal Pengaturan - Standar, AES-128, atau AES-256 - dan atur kata sandi untuk arsip. Kata sandi dibagikan di luar pita; arsip itu sendiri menjadi tidak berguna tanpanya. Pengaturan berada di modal Pengaturan yang sama dengan nama arsip, jadi alur kerja yang sama yang menamai keluaran juga menguncinya. File diunggah melalui HTTPS ke AWS untuk langkah pembangunan ZIP, hasilnya dikirim kembali sebagai unduhan, dan file sumber otomatis dihapus dari server setelah jendela retensi singkat.

Apa yang tidak disusutkan ZIP

Jika folder yang ingin Anda gabungkan adalah foto dan video, harapkan pengurangan ukuran yang sangat kecil. JPG, PNG, MP4, MP3 sudah dikompresi oleh codec mereka sendiri, jadi pembungkus ZIP di sekelilingnya menyusutkannya sebesar 0 hingga 5% dalam banyak kasus. Susutkan media terlebih dahulu jika tujuannya adalah pengiriman yang lebih kecil: buka kompres gambar atau konverter video, jalankan pengurangan ukuran per-file, lalu gabungkan file yang dioptimalkan ke dalam satu ZIP. Untuk serah terima proyek dan pengiriman multi-file - kode sumber, log, CSV, teks biasa, gambar tidak terkompresi - zip massal dalam satu unggahan untuk mendapatkan arsip yang lebih kecil dan satu lampiran, yang lebih disukai sebagian besar widget unggah daripada beberapa file.

Memilih di mana melakukan ZIP: panduan ini vs halaman alat

Pembuat berbasis browser di /zip-tools/zip-file.html adalah tempat arsip sebenarnya dibangun. Panduan ini adalah halaman perutean-dan-konteks: ia menamai tugas pembaca ("kompres file menjadi satu ZIP"), menunjuk ke pembuat, dan menampilkan keputusan trade-off ukuran dan kata sandi yang mungkin ingin Anda buat sebelum mengunggah. Panduan saudara di /guides/make-zip-file-online.html mencakup alur kerja yang sama dengan kata-kata "buat file zip", dan /guides/compress-zip.html masuk lebih dalam ke sudut trade-off ukuran. Ketiga panduan menyatu pada pembuat yang sama.
