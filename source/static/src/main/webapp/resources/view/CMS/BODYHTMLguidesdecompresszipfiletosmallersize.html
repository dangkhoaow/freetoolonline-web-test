Eine ZIP-Datei Kleiner Komprimieren

Buendeln Sie eine einzelne Datei, eine Mehrfach-Auswahl oder einen ganzen Ordner (Unterordner erhalten) zu einem ZIP-Archiv, und die resultierende Datei ist nur dann kleiner als die Summe ihrer Eingaben, wenn die Eingaben komprimierbar sind: einfacher Text, Logs, CSV-Exporte, JSON-Dumps, Quellcode und unkomprimierte Bilder. Der Browser-seitige Ersteller unter /zip-tools/zip-file.html laedt jede Eingabe ueber HTTPS hoch, erstellt das Archiv und gibt eine downloadbereite ZIP zurueck - keine Installation, kein Konto, keine Kopie nach dem Aufbewahrungsfenster.

Bereits-komprimierte Eingaben aendern das Bild. Als JPG gespeicherte Fotos, als MP4 gespeicherte Videos, als MP3 gespeicherte Audios und die meisten PDFs wurden bereits im DEFLATE-Stil komprimiert, als sie geschrieben wurden, sodass das Verpacken in einer ZIP nur etwa null bis fuenf Prozent gewinnt - manchmal wird das Archiv aufgrund des Verpackungs-Overheads pro Eintrag sogar geringfuegig groesser. Wenn Ihr Ziel die kleinstmoegliche Lieferung ist, verkleinern Sie die Medien zuerst mit einem Bild- oder Video-Kompressor und buendeln Sie dann die optimierten Dateien zusammen.
