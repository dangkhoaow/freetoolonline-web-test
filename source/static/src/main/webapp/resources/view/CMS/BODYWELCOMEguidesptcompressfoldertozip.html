O leitor que chegou aqui procurando "comprimir pasta zipada" geralmente ja tem o .zip pronto e quer torna-lo menor antes de enviar por email ou subir para um servico de transferencia. O caminho mais curto e o compactador ZIP com nivel de compressao maximo; o caminho que entrega resultado real e descompactar primeiro, reduzir as imagens individuais com o compressor de imagens, e depois re-zipar.

A resposta honesta (honest answer) para "como faco essa pasta zipada ficar menor": re-compressing a `.zip` almost never shrinks it, because the ZIP deflate algorithm already removed the byte-level redundancy on the first pass. Rodar o mesmo algorithm numa cadeia de bytes que ja parece aleatoria nao acha mais nada para shrink; the result fica dentro de 1% do tamanho do `.zip` original.
