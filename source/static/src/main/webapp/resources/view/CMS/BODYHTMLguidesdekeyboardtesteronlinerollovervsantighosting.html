Tastatur-Test - Rollover vs Anti-Ghosting

Zuletzt geprueft 2026-04-27. Oeffnen Sie den Tastatur-Test, um Ihre Tastatur im Browser zu pruefen - keine App-Installation erforderlich.

30-Sekunden-Antwort. N-Key-Rollover (NKRO) ist, wie viele Tasten gleichzeitig gehalten werden koennen und trotzdem korrekt registrieren. Anti-Ghosting ist die zugehoerige Loesung, die Phantom-Tastendrucke unterdrueckt, wenn mehrere physische Tasten gehalten werden. Es sind verschiedene Probleme: NKRO geht um fehlende Tasten, Ghosting um zusaetzliche. Ein Tastatur-Test im Browser zeigt beides - mehrere Tasten druecken, sehen, was registriert.

Was Ghosting tatsaechlich ist

Die meisten guenstigen Tastaturen verdrahten die Tasten in einer Matrix - Reihen und Spalten, die der Controller scannt. Wenn drei Tasten an den Ecken eines Rechtecks in der Matrix gehalten werden, kann der Controller eine vierte Taste an der ungenutzten Ecke lesen, die nie gedrueckt wurde. Diese vierte Taste ist ein "Geist". Die Loesung ist Anti-Ghosting-Schaltung (zusaetzliche Dioden pro Taste) oder ein anderes Scan-Protokoll; so oder so weigert sich die Tastatur, den Phantomdruck zu registrieren.

So testen: drei Tasten halten, die sich eine Reihe und Spalte in der Matrix teilen. WASD plus Q ist der Standard-Gaming-Test - auf einer ghosting-anfaelligen Tastatur laesst das Halten von W, A, S waehrend Q gedrueckt wird Q ausfallen oder eine vierte Taste erscheinen. Auf einer ordentlich anti-ghost Tastatur registriert jeder Druck korrekt.

Was Rollover bedeutet

N-Key-Rollover ist eine Spezifikation, kein Ausbleiben von Versagen. NKRO bedeutet, dass jede Taste korrekt registriert, unabhaengig davon, wie viele gleichzeitig gehalten werden. 6-Key-Rollover (6KRO) bedeutet, dass bis zu 6 Tasten plus Modifikatoren registrieren; der 7. Druck wird verworfen. 2KRO ist die guenstigste Spezifikation und erscheint auf Billig-Tastaturen - sobald Sie eine dritte Taste halten, gehen Dinge kaputt.

Zum Tippen ist sogar 2KRO in Ordnung - die meisten Leute halten nicht drei Tasten gleichzeitig. Fuer Gaming ist NKRO die richtige Wahl. Fuer Musikproduktion mit vielen gleichzeitigen Noten ist NKRO erforderlich.

Wie im Browser zu testen

  1. Oeffnen Sie den Tastatur-Test. Jede physische Taste erscheint auf der Bildschirmtastatur.
  2. Halten Sie WASD mit einer Hand und die Leertaste mit der anderen. Der Test sollte alle fuenf Tasten hervorgehoben zeigen, jede einzelne.
  3. Halten Sie nun WASD plus Shift, Ctrl und eine Buchstabentaste wie Q gleichzeitig. Modifikatortasten haben oft eine separate Matrix-Verdrahtung; dies prueft, ob Gaming-stil Akkordeingaben funktionieren.
  4. Loslassen; die Hervorhebungen sollten verschwinden. Eine Taste, die nach dem Loslassen "gedrueckt" bleibt, deutet auf einen klemmenden Schalter oder ein Debounce-Problem im Controller hin.

Was der Browser-Test nicht pruefen kann

Das OS sitzt zwischen Tastatur und Browser. Einige Dinge, die der Browser nicht direkt sehen kann:

  • USB-Berichtsrate. Tastaturen berichten mit 125 Hz, 500 Hz oder 1000 Hz. Der Browser sieht Druckereignisse, wenn sie eintreffen, kann Intervalle bei dieser Aufloesung aber nicht aussagekraeftig messen.
  • Hardware-Tastenneuzuweisung. Manche Tastaturen (Apple, Gaming mit Onboard-Speicher) weisen Tasten auf der Firmware-Ebene neu zu. Der Browser sieht die neu zugewiesene Taste, nicht die physische.
  • Funkstoerung. Bluetooth und 2,4-GHz-Funk koennen Tasten unter Interferenz fallen lassen oder wiederholen. Der Test wird offensichtliche Ausfaelle zeigen, kann das Radio aber nicht diagnostizieren.
