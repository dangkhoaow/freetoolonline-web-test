ZIP is not always the right wrapper. For a single document you want to keep editable, re-save the file from the PDF tools instead of compressing it. For tamper-evident bundles where the recipient needs to confirm nothing changed in transit, pair the archive with an MD5 or SHA hash from the hashing tools. And note the limits: this toolset cannot crack unknown passwords, and it does not handle RAR, 7z, or TAR.GZ - ask the sender to re-archive as ZIP if those land in your inbox.
