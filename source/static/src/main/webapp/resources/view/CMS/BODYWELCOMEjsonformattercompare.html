JSON Compare Online

Last reviewed: 2026-05-18

Reviewed by FreetoolOnline editorial team

Paste two JSON documents side by side and read which keys differ, which are added, and which are removed. The diff runs in your browser - no upload, no account, no installs - and reports each difference keyed by its JSON path (for example, $.users[2].email for a value change inside an array).

When to use this tool. Two API responses you want to A/B against each other. Two config files you suspect drifted. Two payloads from staging and prod where one is silently missing a field. Anywhere a single-document formatter (such as /json-formatter.html or /json-formatter-editor.html) is not enough because the question is "what changed between A and B".

What you get. A per-key diff list ordered by JSON path. Keys that were added in B but missing in A, keys that were removed in B but present in A, and keys that exist in both but hold different values. The comparator runs on input event with a 250 ms debounce; there is no Compare button to click.

What this tool does not do. No JSON schema validation, no schema generation, and no large-file streaming - the in-memory comparator is bounded by the browser's JSON.parse capability. For free-form text diffs against non-JSON payloads, use /text-diff.html instead.
