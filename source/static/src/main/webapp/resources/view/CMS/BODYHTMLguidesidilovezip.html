Pilih tujuan yang tepat untuk apa yang sebenarnya perlu Anda lakukan

"Alat zip online" sebenarnya adalah tiga tugas yang berbagi satu nama: membangun arsip baru, membuka yang sudah ada, atau menyusutkan yang sudah ada. Jalur pembangunan adalah yang paling umum - Anda memiliki folder atau set lampiran dan Anda ingin satu .zip yang dapat Anda serahkan tanpa menginstal pengarsip desktop terlebih dahulu. Buka /zip-tools/zip-file.html, jatuhkan folder ke zona unggah, beri nama arsip di modal Pengaturan, opsional tambahkan kata sandi, klik Convert, dan unduh hasilnya. Jalur pembukaan melewati /zip-tools/unzip-file.html: jatuhkan .zip ke halaman, dan halaman akan mencantumkan kontennya sehingga Anda dapat memilih file mana yang akan diekstrak kembali ke mesin Anda.

Apa arti "online" di sini

"Online" di halaman ini berarti pembuat berjalan di layanan pembangun pribadi yang dihosting oleh FreetoolOnline, bukan sebagai program yang diunduh. File meninggalkan perangkat Anda melalui HTTPS selama beberapa detik pembangunan berlangsung, kemudian file sumber dihapus dari layanan setelah jendela retensi singkat. .zip itu sendiri ditawarkan kembali ke browser Anda dan tidak disimpan di layanan setelah unduhan dimulai. Tidak ada akun dan tidak ada pengiriman email. Trade-off dibandingkan dengan pengarsip sepenuhnya di sisi browser adalah keandalan pada folder besar: layanan pembangun menangani input yang melampaui batas memori browser di mana pengarsip JavaScript-saja akan macet atau berhenti.
