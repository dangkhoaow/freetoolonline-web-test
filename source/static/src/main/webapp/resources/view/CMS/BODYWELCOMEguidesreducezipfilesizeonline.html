If your archive needs to travel through a password-protected channel, the optional password setting on the zip tool applies AES encryption at build time, so the recipient can open the file with a shared secret rather than a separate transfer link. The same flow handles a single folder, a stack of loose files or a previously-built zip - the output is one archive in all three cases, and the recipient extracts it back to the original folder layout on their machine.
