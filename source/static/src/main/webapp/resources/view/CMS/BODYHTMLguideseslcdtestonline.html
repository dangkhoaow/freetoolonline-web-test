Test LCD

Una prueba LCD muestra colores solidos a pantalla completa - rojo, verde, azul, blanco y negro - para que el sangrado de luz, los pixeles atascados y la iluminacion irregular se vean contra un fondo uniforme en segundos. Abre la herramienta, toca cada muestra de color para llenar toda la pantalla y busca cualquier punto, linea o area que no coincida con el color circundante.

Tambien llamado prueba de pantalla o test de display, este chequeo LCD es el mismo diagnostico rapido: mostrar un color solido a la vez para que los pixeles muertos o atascados y el sangrado de luz destaquen contra un fondo uniforme.
