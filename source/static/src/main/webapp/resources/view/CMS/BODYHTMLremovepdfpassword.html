What this tool does: takes a PDF you have permission to open plus the password you already know, and returns a copy of the same PDF without the password prompt. It accepts the open password (also called the user password) and the owner password (also called the permissions password).

What it does NOT do: recover, brute-force, or guess a forgotten password. If you do not know the password, the tool returns an error and the file stays encrypted.
