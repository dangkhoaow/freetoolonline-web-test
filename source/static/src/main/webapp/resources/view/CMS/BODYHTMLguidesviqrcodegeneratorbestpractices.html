Bo Tao QR Code - Cac Thuc Hanh Tot Nhat

Lan ra soat cuoi 2026-04-27. Mo bo tao QR code de tao code trong trinh duyet ban ma khong gui payload di dau.

Cau tra loi 30 giay. Mot QR code luon quet duoc phu thuoc vao nam cai dat. Sua loi muc M hoac cao hon (cho code song sot tu hu hai nhe). Tuong phan cao (den tren trang, khong gradient nhe). Kich thuoc vat ly toi thieu 2 cm (nho hon that bai tren camera dien thoai). Vung yen tinh it nhat bon modules quanh canh. Va payload ngan - URL dai tao code day, kho quet; dung URL shortener truoc.

Sua loi - cai dat bi bo qua nhieu nhat

QR code co bon muc sua loi. Chung noi voi bo ma hoa bao nhieu cua code co the bi hu hai hoac che truoc khi quet that bai:

  • L (Low). Khoi phuc den ~7% hu hai. Code nho nhat, it khoan dung loi nhat. Dung chi khi code o tren man hinh va khong bi xuoc.
  • M (Medium). Khoi phuc ~15% hu hai. Mac dinh trong hau het bo tao. Tot cho hau het ngu canh in.
  • Q (Quartile). Khoi phuc ~25% hu hai. Dung khi code co the chong logo hoac bi che mot phan.
  • H (High). Khoi phuc ~30% hu hai. Lua chon dung cho code in tren bien hieu ngoai troi, tren bao bi, hoac noi mong doi su hao mon.

Danh doi: sua loi cao hon lam code day hon (nhieu o vuong den-trang hon moi vung) cho cung mot payload. Chon muc thap nhat ma moi truong cua ban khoan dung.

Tuong phan va mau sac

Tieu chuan la tien canh toi tren nen sang. Den tren trang quet tot nhat. Dao nguoc (sang tren toi) thuong that bai vi scanner camera dien thoai gia su tinh phan cuc tieu chuan.

Mau tuy chinh hoat dong neu ty le tuong phan giu tren 4:1 - mot xanh dam tren trang la on, mot xam trung binh tren trang khong on. Sang-tren-toi khong dang tin cay; khong bao gio dung QR trang-tren-den cho vat lieu in. Nen gradient pha quet hoan toan; khu vuc code phai la mot mau ran duy nhat.

Kich thuoc vat ly va vung yen tinh

Kich thuoc toi thieu quet duoc phu thuoc khoang cach. Quy tac ngo: do dai canh code phai it nhat 1/10 khoang cach quet. Mot code quet tu 30 cm (mot dien thoai tren tay) can it nhat 3 cm. Mot code quet tu 3 m (mot poster) can it nhat 30 cm.

Vung yen tinh la khoang trang quanh code. Dac ta QR yeu cau it nhat bon modules (cac o vuong nho) le tren ca bon canh. Dat code sat van ban, mot vien, hoac mot do hoa khac lam scanner that bai du chinh code con nguyen ven.

Do dai payload quan trong hon ban nghi

Do day cua mot QR code tang voi kich thuoc payload. Ma hoa URL 10 ky tu tao code thua ma quet tu khoang cach mot met. Ma hoa URL 200 ky tu voi tracking parameters tao code day can lon hon vat ly va quet tu gan hon.

Luon rut ngan URL truoc. example.com/p tot hon example.com/products/winter-collection?utm_source=poster&utm_campaign=2026-q1. Redirect co the mang tracking parameters phia server; QR mang chi link ngan. Ket qua la mot code nho hon, dang tin cay hon.

Test truoc khi in

In code o kich thuoc ban du dinh dung, quet tu khoang cach va goc ban mong, va thu it nhat hai dien thoai - mot moi, mot vai nam tuoi. Mot code quet duoc tu 5 cm nhung that bai tu 50 cm la kich thuoc sai. Bo tao QR code xem truoc code o nhieu kich thuoc truoc khi tai.
