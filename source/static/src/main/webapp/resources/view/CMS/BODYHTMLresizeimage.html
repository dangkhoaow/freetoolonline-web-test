Image resizer scales a PNG or JPG to a specific width and height in pixels while keeping the aspect ratio locked.

  1. Upload a PNG or JPG image.
  2. Enter a target max width and max height in pixels; the tool scales down without distortion.
  3. Download the resized image.

Tip: For sharp icon work, start from a vector source instead of resizing a raster up; bicubic interpolation handles photographs cleanly but cannot recover lost detail.
