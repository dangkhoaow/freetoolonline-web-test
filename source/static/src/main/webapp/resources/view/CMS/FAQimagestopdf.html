Can I control page size, orientation, and margins in the output PDF?

Yes - the settings panel exposes page size (A4, US Letter, plus several photo-book and magazine presets), orientation (portrait / landscape), and margin width. Default is A4 portrait with a small margin, which suits most sharing and printing use cases; switch to landscape when most images are wider than tall, and pick a larger margin only if the document will be bound or three-hole-punched.

Does the tool preserve image EXIF data in the final PDF?

EXIF metadata (camera model, shutter speed, capture date, GPS if present) carries over into the PDF as document-level metadata for each embedded image page. If you are assembling photos for archival and need to strip GPS coordinates before sharing publicly, uncheck the EXIF option in settings before converting - the PDF then lands without location data.

Should I merge photos into one PDF or use a portfolio / album format?

Use this tool when readers will consume the photos in a fixed reading order (event recap, portfolio, photo report, print-ready album). If recipients need to open specific photos independently, export them as a ZIP of JPG files instead - a merged PDF forces linear browsing. For the reverse direction (extracting photos out of an existing PDF), see PDF to Images.
