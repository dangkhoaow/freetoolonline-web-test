If neither route opens the archive cleanly, the file may not be a true ZIP - check the extension; common look-alikes are .rar, .7z, and .tar.gz, each of which needs a different extractor. The plain-ZIP page lists those formats in its supported list, but for safety re-package the archive as .zip on the sender's side when in doubt.
