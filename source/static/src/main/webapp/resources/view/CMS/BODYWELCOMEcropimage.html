Crop an image online for free without installing anything. Drag the crop box anywhere on the upload and pull the corners until the frame matches the target. Five aspect-ratio presets (16:9, 4:3, 1:1, 2:3, Free) cover the common social sizes and avoid manual pixel math; for a custom rectangle pick Free and type the X, Y, Width, and Height into the side inputs to land an exact pixel region. The original stays in the workspace, so a second crop on a different ratio takes one click instead of a re-upload.

Every crop on this page runs inside the browser via HTML canvas - the picture is never uploaded to a server, never shared with a third-party processing host, and never persists past the page reload. The Download button writes the cropped result straight into the browser's download folder; nothing about the file leaves the device, and closing the tab is enough to drop the working copy. Useful when the source picture contains a face, a personal document, or other sensitive content that should not transit a remote service.

The crop output preserves the source format. A JPG comes out as a JPG, a PNG comes out as a PNG, a WebP comes out as a WebP - no automatic re-encoding to a different codec, no transparency loss on PNG, no quality re-compression on JPG. Useful when the workflow downstream expects a specific extension: a forum-upload rule that only accepts JPG, a CMS uploader that rejects WebP, a social-network format constraint, or a print-shop file-format spec. If the next step in the pipeline is to switch container format, that conversion is a separate trip to image converter tools; this page only handles the framing-and-cut decision and keeps the input format intact.

Need to straighten a sideways shot before the crop? The Rotate Left and Rotate Right buttons turn the working image in 45-degree steps without leaving the page, so a landscape phone photo captured rotated lands upright with one click. For a tilted horizon, drag the rotation handle for a free-angle correction; the crop frame stays anchored to the rotated content so the final cut tracks the new orientation. The drag-mode toggle switches between Move - to pan a zoomed-in canvas - and Crop, the default drag-to-select selection mode that picks up where the rotation finishes.
