When the goal is sending the smallest possible delivery, split the upload before you archive. Zip the compressible inputs together - text files, CSVs, source code, uncompressed images - for the real size win, then handle the media separately: resize, re-encode, or leave them as-is. The resulting two-package delivery (a small ZIP of text-like content plus the media untouched) is usually smaller than one ZIP of everything mashed together.
