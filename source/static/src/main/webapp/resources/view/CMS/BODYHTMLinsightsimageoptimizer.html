Upload an image and the optimizer audits it locally to recommend an optimized JPEG quality target between 40 and 85 for faster page-load delivery. The recommended preview appears in the grid above once the analysis completes, and the quality slider in Settings lets you adjust the target before downloading the optimized output. The original file returns unchanged from the server, and the optimizer does not convert format to WebP or AVIF or batch-reprocess a whole directory in one upload - reach for a dedicated converter or a batch tool when those are the steps you actually need.
