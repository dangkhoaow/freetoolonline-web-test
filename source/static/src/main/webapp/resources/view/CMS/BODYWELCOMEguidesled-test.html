Three live readouts on the implementing tool at /lcd-test.html show the current viewport size, the OS-reported screen resolution, and the native screen resolution multiplied by devicePixelRatio. The readouts are useful for confirming retina or hi-DPI scaling before declaring a suspect dot dead: a 100% pixel-perfect uniform color across a high-DPI panel looks subtly different on a 200%-scaled display than on a 100%-scaled one, and the readout numbers explain why without forcing the reader to dig into OS display settings. The test runs entirely in the browser - no upload, no server round trip, no install required - so it continues working on a managed laptop or an air-gapped device where third-party tools cannot be installed.
