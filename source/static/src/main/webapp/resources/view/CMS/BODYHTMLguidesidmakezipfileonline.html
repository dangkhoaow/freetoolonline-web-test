Buat File ZIP Online

Membuat file ZIP online berarti mengambil satu file, beberapa file, atau seluruh folder (subfolder dipertahankan) dan menulisnya sebagai satu arsip .zip yang dapat Anda kirim melalui email, unggah, atau cadangkan sebagai satu lampiran tunggal. Pembuat di browser di /zip-tools/zip-file.html melakukan persis hal itu: Anda jatuhkan file, halaman membundelnya di sisi server melalui HTTPS, file sumber dihapus setelah jendela retensi singkat, dan arsip yang dihasilkan kembali sebagai unduhan.

Apa yang sebenarnya dilakukan pembuat ZIP online

Pembuat menangani tiga tugas dalam satu halaman. Pertama, ia membundel: pilihan multi-file dan seluruh folder (subfolder dipertahankan) berubah menjadi satu .zip yang diunduh penerima dengan sekali klik. Kedua, ia dapat mengunci dengan kata sandi: kata sandi opsional dan metode enkripsi (Standard, AES-128, atau AES-256) menulis arsip yang dilindungi ketika kata sandi akan dibagikan melalui saluran terpisah. Ketiga, ia memberi nama keluaran: jendela Pengaturan memungkinkan Anda menetapkan nama file arsip sebelum diunduh sehingga penerima melihat proyek atau topik dalam nama file alih-alih cap waktu generik.

Kapan membuat ZIP benar-benar menghemat ruang

ZIP menyusutkan masukan hanya ketika masukan tersebut dapat dipadatkan. Teks polos, log, CSV, JSON, kode sumber, dan bitmap yang tidak terkompresi biasanya turun ke 30-60% dari ukuran aslinya saat dizip bersama karena byte-nya berulang dan algoritma ZIP menghilangkan pengulangan. Foto dalam JPG, video dalam MP4, audio dalam MP3, dan sebagian besar PDF sudah dipadatkan oleh codec mereka sendiri, sehingga ZIP membungkusnya dengan perubahan ukuran sekitar 0-5% dan dapat sedikit bertambah karena overhead pengemasan. Trik untuk pengiriman yang lebih kecil adalah mengezip file yang dapat dipadatkan bersama untuk mendapatkan keuntungan ukuran yang sebenarnya dan menangani media secara terpisah (mengubah ukuran atau menyandikan ulang), bukan mengezip semuanya dan berharap.

Cara membuat file ZIP di browser

  1. Buka /zip-tools/zip-file.html.
  2. Jatuhkan atau pilih file (folder juga bisa; subfolder dipertahankan).
  3. Opsional: buka Pengaturan, tetapkan nama file, dan tambahkan kata sandi plus metode enkripsi bila arsip perlu melewati saluran di mana kata sandi dibagikan terpisah.
  4. Klik Zip. Halaman mengunggah masukan melalui HTTPS, membangun arsip di sisi server, dan menampilkan tautan unduhan.
  5. Simpan .zip. Jika Anda menetapkan kata sandi, kirim melalui saluran terpisah dari file.
