Como funciona la extraccion

Elige primero el formato de salida. PNG mantiene fidelidad pixel a pixel y transparencia, que es lo que quieres cuando el frame se convierte en una imagen estatica de publicacion o un activo de boton. JPG produce archivos mas pequenos y es la mejor opcion cuando el frame entra en una presentacion o un articulo donde el ancho de banda importa mas que cada ultimo pixel. La configuracion de FPS controla cuantos frames por segundo del GIF de origen guarda el extractor en el disco - menor FPS significa menos frames, mas espaciados; mayor FPS significa mas frames, agrupados mas cerca. Si solo necesitas un frame heroe para redes sociales, 10 FPS es suficiente; para frames clave de animacion o trabajo de storyboard, 24 o 60 captura el movimiento intermedio.

Que recibes de vuelta

Despues de que la extraccion termine, los frames aparecen en una tira de vista previa para que puedas recorrerlos antes de descargar. Puedes tomar cualquier frame individual, o descargar todo el conjunto de una vez. El nombrado es secuencial para que los frames se alineen con el orden de reproduccion del GIF de origen - util cuando estas eligiendo el frame de pico de movimiento para una miniatura o cosiendo un subconjunto de vuelta en otra herramienta. Los archivos se procesan en nuestros servidores y se eliminan automaticamente despues de que expire la ventana de retencion.
