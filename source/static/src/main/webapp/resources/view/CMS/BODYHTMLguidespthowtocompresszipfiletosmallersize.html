Como Comprimir um Zip para um Tamanho Menor

Ultima revisao: 2026-04-27. Abra a ferramenta zip para re-comprimir no navegador sem upload do arquivo.

Resposta de 30 segundos. Re-zipar um zip existente quase nunca o torna menor porque os conteudos ja estao comprimidos. As duas coisas que funcionam: extraia primeiro, comprima os arquivos mais pesados dentro (imagens, PDFs, audio) com uma ferramenta diferente, depois re-zipe; ou escolha um formato mais forte (7z, zip-com-LZMA) quando o destinatario suportar.

Por que re-zipar nao faz quase nada

Zip usa o algoritmo Deflate. Deflate encontra padroes de bytes repetidos e os substitui por codigos mais curtos. Apos uma passagem, os padroes desaparecem. Uma segunda passagem nos mesmos dados nao encontra nada para substituir.

O que funciona: extrair, comprimir, re-zipar

  1. Extraia o arquivo localmente para que voce possa ver os arquivos individuais.
  2. Execute o compressor certo nos pesados. Um JPEG de 5 MB cai para menos de 1 MB via compressao de imagem.
  3. Re-zipe os arquivos menores. Use compressao Best em documentos e Store na midia (agora menor).
