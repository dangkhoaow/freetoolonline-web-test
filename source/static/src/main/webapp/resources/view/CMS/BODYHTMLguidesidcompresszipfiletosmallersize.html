Kompres File ZIP ke Ukuran Lebih Kecil

Bundel satu file, pilihan multi-file, atau seluruh folder (sub-folder dipertahankan) menjadi satu arsip ZIP, dan file yang dihasilkan lebih kecil dari jumlah inputnya HANYA ketika input bisa dikompresi: teks polos, log, ekspor CSV, dump JSON, kode sumber, dan gambar tidak terkompresi. Pembuat sisi browser di /zip-tools/zip-file.html mengunggah setiap input melalui HTTPS, membangun arsip, dan mengembalikan satu ZIP siap unduh - tanpa instal, tanpa akun, tanpa salinan disimpan setelah jendela retensi.

Alat implementasi/zip-tools/zip-file.html

Input yang sudah-terkompresi mengubah gambarannya. Foto disimpan sebagai JPG, video disimpan sebagai MP4, audio disimpan sebagai MP3, dan sebagian besar PDF sudah dikompresi gaya DEFLATE saat ditulis, jadi membungkusnya di dalam ZIP hanya menghemat sekitar nol hingga lima persen - kadang arsipnya berakhir sedikit lebih besar karena overhead pengemasan per entri. Jika tujuan Anda adalah pengiriman terkecil yang mungkin, perkecil media terlebih dahulu dengan kompresor gambar atau video, lalu gabungkan file yang dioptimalkan bersama.
