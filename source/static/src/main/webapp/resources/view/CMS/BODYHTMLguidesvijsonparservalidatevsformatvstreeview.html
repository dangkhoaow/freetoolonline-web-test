JSON parser online: kiem tra vs dinh dang vs vista cay - tinh nang nao ban can?

Lan ra soat cuoi 2026-05-05. JSON parser mien phi tren trang nay gop ba tinh nang con vao mot trang: kiem tra (bat loi phan tich), dinh dang / lam dep (chuyen mot blob mot dong da giam thieu thanh JSON co tang va de doc), va vista cay (kham pha cau truc tuong tac). Ba tac vu doc gia khac nhau, ba diem khoi dau khac nhau. Huong dan nay nau ten trieu chung moi cai bat va thu tu su dung khi nhieu hon mot ap dung.

Cau tra loi 30 giay. Neu JSON cua ban dang nem loi phan tich trong code hoac trinh chinh sua to do, kiem tra truoc - parser revela vi tri loi dau tien. Neu JSON cua ban la mot dong dai da giam thieu sao chep tu tab mang, dinh dang no - tang lam cho phan hoi 50 dong de doc khi diff. Neu JSON cua ban phan tich sach se nhung ban khong tim duoc khoa chon trong cac doi tuong long, dung vista cay - moi thuoc tinh tro thanh mot dong ban co the mo rong va kiem tra. Khi nhieu hon mot ap dung, thu tu luon la kiem tra → dinh dang → vista cay: loi phan tich chan dinh dang, va JSON khong dinh dang kho doc o dang cay.

Khi nao kiem tra truoc

JSON parser kiem tra khi ban dan. Neu input cua ban khong hop le, panel ben phai hien thong bao do Invalid JSON va chi vao loi dau tien. Bon nguyen nhan thuong gap nhat la thieu ngoac ({ khong co } tuong ung), dau phay cuoi ({"a": 1,} - JSON khong cho phep), dau ngoac kep khong thoat trong gia tri chuoi ("key": "she said "hi""), va comment di lac (khoi // hoac /* */ - JSON khong ho tro comment). Sua loi dau tien, dan lai, va parser cho ban biet con loi nao khong. Kiem tra chay truoc dinh dang vi formatter khong the tang JSON ma no khong the phan tich.

Khi nao dinh dang (lam dep)

Tinh nang con dinh dang / lam dep chuyen mot chuoi JSON mot dong da giam thieu thanh tai lieu duoc tang dung voi moi thuoc tinh tren dong cua no. Dung khi ban sao chep phan hoi tu tab mang, dau ra curl, hoac log server va can doc hoac diff. Nut Copy Beautified JSON tra ve cung du lieu voi tang de ban co the dan vao review code, ticket, hoac fixture test don vi. Dinh dang khong thay doi du lieu - cac khoa, gia tri, va hinh dang giong het input - chi khoang trang va xuong dong thay doi. Neu ban muon huong nguoc lai (chuyen JSON da lam dep ve mot dong de gui di), cung parser giu ca hai dang dong bo.

Khi nao dung vista cay

Tinh nang con vista cay render JSON da phan tich thanh cay tuong tac trong panel ben phai. Moi thuoc tinh la dong rieng; doi tuong long mo rong va sup; mang hien cac item theo chi muc. Dung khi JSON phan tich sach se nhung ban khong tim duoc khoa chon bon cap sau, hoac khi ban muon kiem tra hinh dang cua mang - moi item co cung loai, mot so item thieu truong, chi muc co khop voi cai API tai lieu? Bat Show Types de hien loai cua moi gia tri (string, number, object, array, boolean, null) ben canh khoa cua no. Bat Show Indexes de gan nhan cho moi item mang voi vi tri base-zero. Vista cay chi doc - khong sua doi JSON trong panel input.

Khi nao dung nhieu hon mot tinh nang

Ba tinh nang xep theo thu tu co dinh: kiem tra truoc (neu khong formatter va vista cay that bai), roi dinh dang (de phan hoi 200 dong de doc), roi vista cay (khi van ban da dinh dang van qua dai de scan). Luong cong viec thong dung khi debug API ben thu ba: dan phan hoi tho, sua loi phan tich dau tien ma validator hien, nhap Copy Beautified JSON de co ban sao de doc cho ticket bug, va sau do dung vista cay de chi vao khoa cu the ma nguoi tieu thu API mong doi. Cho cac quyet dinh khac cua cong cu phat trien khong lien quan trong danh muc developer-tools, xem cac huong dan so sanh lien ket ben duoi.

Cau hoi thuong gap

Vista cay co sua doi JSON cua toi khong?

Khong. Vista cay la render chi-doc cua input da phan tich; bat Show Types hoac Show Indexes chi anh huong cach cay duoc hien thi, khong phai JSON co ban. De thay doi JSON, chinh sua panel input ben trai.

Su khac biet giua minify va beautify la gi?

Minify xoa khoang trang, xuong dong, va cac ky tu khong y nghia khac de lam JSON nho nhat co the (de van chuyen hoac luu tru). Beautify (cung goi la "pretty-print" hoac "dinh dang") them khoang trang va xuong dong de con nguoi co the doc. Du lieu giong het trong ca hai dang - chi dinh dang thay doi. Tuong tu CSS cho code dung hai cong cu rieng biet: CSS minifier va CSS unminifier.

Toi co the dan JSON5, JSON-with-comments, hoac NDJSON khong?

Chi JSON nghiem ngat. JSON5 (cho phep comment, dau phay cuoi, va chuoi voi dau ngoac don), JSON-with-comments (doi khi goi la JSONC, dung boi VS Code), va NDJSON (mot doi tuong JSON moi dong) deu that bai kiem tra vi chung khong phai JSON nghiem ngat. Bo cac tinh nang khong chuan truoc, hoac chuyen NDJSON thanh mang JSON, roi dan.

Co gi duoc gui den server khong?

JSON parser chay trong trinh duyet cua ban; cac luot kiem tra, dinh dang, va vista cay thuc thi tren client ma khong upload JSON len server. Dieu nay huu ich khi JSON chua du lieu nhay cam va ban khong muon no roi thiet bi.
