So Pruefen Sie die Kameraqualitaet Ihres Telefons

Zuletzt geprueft: 2026-04-27. Oeffnen Sie den Kameratest, um die Kamera Ihres Telefons im Browser zu pruefen - ohne App-Installation.

30-Sekunden-Antwort. Fuenf Tests erfassen fast jedes Kameraproblem: bestaetigen Sie, dass die Live-Vorschau sich oeffnet (Kamera waehlbar und Browser hat Berechtigung), pruefen Sie die Aufloesung gegen das Datenblatt, fokussieren Sie auf Text auf Armlaenge und pruefen Sie die Schaerfe, machen Sie ein Foto bei hellem und schwachem Licht und pruefen Sie sichtbare Flecken auf der Linse.

Test 1 - die Kamera oeffnet sich

Der erste Fehlerfall ist "Kamera erscheint nicht im Browser". Zwei haeufige Ursachen: der Browser hat keine Kameraberechtigung, oder eine andere App haelt die Kamera.

Test 2 - Aufloesung passt zur Spezifikation

Telefonkamera-Spezifikationen werden in Megapixeln beworben. Ein 12-MP-Sensor produziert 4032x3024-Fotos. Die Testseite zeigt die tatsaechliche Aufloesung, die der Browser erhaelt.

Test 3 - Fokus auf Text

Halten Sie gedruckten Text auf Armlaenge. Tippen Sie zum Fokussieren. Der Text sollte in 0,5 Sekunden scharf werden. Langsamer Fokus oder weiche Vorschau weisen auf defekten Autofokus oder verschmierte Linse hin.
