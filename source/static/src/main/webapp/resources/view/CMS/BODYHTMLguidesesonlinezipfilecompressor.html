ZIP sigue siendo el valor predeterminado seguro para compartir entre plataformas: todo sistema operativo moderno abre .zip nativamente, asi que el destinatario nunca necesita un visor especial. La extension ZIP64 estira el techo del archivo a 16 EB, lo que significa que el limite practico es el tope de adjunto de correo o cuota de subida del otro lado, no el formato en si. Tres flujos de trabajo aparecen una y otra vez: agrupar un proyecto para un adjunto de correo, bloquear archivos detras de una contrasena antes de enviarlos, y recuperar el contenido de una descarga.
