LCD Test Online

An online LCD test cycles a single solid colour across the full viewport — red, green, blue, black, white, and yellow — so dead, stuck, and discoloured pixels stand out against a uniform background; the same six colours cover screen-test, display-test, monitor-test, and dead-pixel-test searches in one place.

Implementing tool/lcd-test.html

On a desktop browser the optional fullscreen toggle calls the OS-level Fullscreen API so the diagnostic field fills the panel edge-to-edge with no chrome, no tabs, and no taskbar in the way of a suspect pixel. On phones narrower than 821 pixels the page quietly falls back to an in-page tint of the document background, because iOS Safari and most Android browsers block fullscreen on non-user-gesture elements — the same six full-screen colours still wash the visible area, so the reader can spot a stuck or discoloured pixel on a handset without fighting the browser for permission. Alongside the colour wash the tool prints the live viewport size, the reported screen resolution, and the native resolution multiplied by the device pixel ratio for retina or hi-DPI panels, so the reader can confirm the test is running at the panel's true pixel grid before declaring a pixel dead.
