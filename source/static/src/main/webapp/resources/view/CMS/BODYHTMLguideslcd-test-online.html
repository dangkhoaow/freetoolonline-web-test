LCD Test Online

An online LCD test cycles a single solid colour across the full viewport — red, green, blue, black, white, and yellow — so dead, stuck, and discoloured pixels stand out against a uniform background; the same six colours cover screen-test, display-test, monitor-test, and dead-pixel-test searches in one place.

Implementing tool/lcd-test.html

The page reports the live viewport size, the screen resolution, and the native screen resolution (multiplied by the device pixel ratio for retina or hi-DPI panels) so the reader can confirm the test is running at the panel's true pixel grid before declaring a pixel dead. Reset returns the document background to its original state in one click so the reader can leave the diagnostic field without reloading the page.
