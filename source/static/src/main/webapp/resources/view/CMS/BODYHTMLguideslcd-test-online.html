Implementing tool/lcd-test.html

On a desktop browser the optional fullscreen toggle calls the OS-level Fullscreen API so the diagnostic field fills the panel edge-to-edge with no chrome, no tabs, and no taskbar in the way of a suspect pixel. On phones narrower than 821 pixels the page quietly falls back to an in-page tint of the document background, because iOS Safari and most Android browsers block fullscreen on non-user-gesture elements - the same six full-screen colours still wash the visible area, so the reader can spot a stuck or discoloured pixel on a handset without fighting the browser for permission. Alongside the colour wash the tool prints the live viewport size, the reported screen resolution, and the native resolution multiplied by the device pixel ratio for retina or hi-DPI panels, so the reader can confirm the test is running at the panel's true pixel grid before declaring a pixel dead.

How to run an LCD test online

  1. Open the tool at /lcd-test.html.
  2. Tap each color tile and scan the screen for dots that stay bright (stuck) or dark (dead) against a uniform field.
  3. On desktop, toggle Fullscreen for edge-to-edge coverage; on phones, rotate to landscape and raise brightness to make faint discoloration easier to spot.

Frequently Asked Questions

Which colors does the LCD test show?

The tool cycles six solid colors: red, green, blue, black, white, and yellow. A single-color field makes pixel defects easier to see than a photo or a busy webpage.

Does this work on phones and OLED screens?

Yes. The same solid-color sweep works on LCD, OLED, and LED panels, and it's usable on phone, laptop, and desktop browsers. On iOS Safari fullscreen may be limited, but the color field still covers the visible viewport.

What do the resolution readouts mean?

The page shows the current viewport size, the OS-reported screen resolution, and the "native" resolution after applying the device pixel ratio. That helps confirm you're testing the panel's real pixel grid on high‑DPI displays.

Can an online LCD test fix a stuck pixel?

This guide is for diagnosis. If you want to try a stuck‑pixel exercise, you can run the color cycle repeatedly for an extended period, but it's not guaranteed. For warranty documentation, the key step is capturing clear photos of the defect on multiple solid colors.
