Split PDF by range extracts a contiguous page range from a PDF into a new file while preserving text, bookmarks, and internal links.

  1. Upload the source PDF and wait for the page count to appear.
  2. Enter the start page and end page (1 ≤ start ≤ end ≤ total pages).
  3. Click Split and download the extracted page-range PDF.

Note: For splitting every page into its own file, use Split PDF to each page. To combine extracted ranges into a new document, chain with Join PDF from multiple files.
