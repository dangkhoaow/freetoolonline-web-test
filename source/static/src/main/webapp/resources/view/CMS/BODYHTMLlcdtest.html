LCD test (dead pixel test) helps you find stuck pixels, dead pixels, and color issues by showing solid colors in full screen. It’s useful for laptops, monitors, and phones.

  1. Enable full screen mode, then click a color (red/green/blue/black/white/yellow).
  2. Inspect the display for dots that don’t change color or areas with uneven brightness.
  3. Use Reset to exit the test and check your resolution readouts below.

Tip: Set brightness to max and temporarily disable Night Shift / blue-light filters for the most accurate results.
