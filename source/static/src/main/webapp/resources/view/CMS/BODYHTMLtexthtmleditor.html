What this tool does: shows raw HTML on the left and a rendered preview on the right; type or paste markup and the preview updates as you go. The toggle button switches between side-by-side and stacked layouts so you can fit the editors to your screen on mobile or desktop.

What it does NOT do: no save, no file upload, and no server-side processing - everything runs in the browser tab. To minify the HTML before pasting it into a production page, send it through CSS Minifier or JS Minifier; to read or strip text from a PDF, use PDF to Text.
