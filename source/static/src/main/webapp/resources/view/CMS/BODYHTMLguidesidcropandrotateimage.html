Potong dan Putar Gambar

Potong dan putar gambar dalam dua langkah singkat di dalam browser: tarik kotak potong ke bingkai yang Anda inginkan (atau pilih salah satu dari lima preset rasio aspek), lalu putar gambar tegak dengan tombol putar atau pegangan sudut bebas. Tombol Unduh menulis hasilnya dalam format sumber - tanpa upload, tanpa instal.

Direview terakhir: 2026-05-23

Direview oleh tim editorial FreetoolOnline

PropertiNilai
Preset rasio aspek5 (16:9, 4:3, 1:1, 2:3, Bebas)
Langkah rotasi45 derajat per klik; pegangan sudut bebas untuk horizon miring
Format outputFormat sumber dipertahankan (JPG, PNG, WebP - tanpa re-encoding)
Tempat pekerjaan terjadiDi browser Anda pada canvas HTML - file tetap di perangkat
Alat implementasi/crop-image.html

Buka alat implementasi di /crop-image.html dan jatuhkan JPG, PNG, atau WebP Anda ke area upload. Seret kotak potong ke mana saja pada gambar dan tarik sudut sampai bingkai cocok dengan apa yang ingin Anda pertahankan. Lima preset rasio aspek mencakup ukuran sosial dan cetak umum. Untuk wilayah piksel yang tepat, pilih Bebas dan ketik posisi X, posisi Y, Lebar, dan Tinggi ke input samping.

Potong dan rotasi mempertahankan format sumber - JPG keluar sebagai JPG, PNG sebagai PNG, WebP sebagai WebP. Gambar diproses pada canvas HTML di dalam browser; tidak pernah berjalan ke server dan tidak pernah bertahan melewati reload halaman. Menutup tab sudah cukup pembersihan.
