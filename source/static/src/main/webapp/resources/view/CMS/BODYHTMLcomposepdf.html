What this Compose PDF tool does

This Compose PDF tool turns rich-text content — paragraphs, headings, simple formatting — into a downloadable PDF in one step. Type or paste your text into the editor, set the file name, and click Submit; the content is sent through the shared upload pipeline (web.includesProcess = true; mainUploaderUrl = service.us-east-1a.freetool.online/) and server-side processing is dispatched by pageName via ServiceFactory to the per-tool composition service. Custom formatting options are forwarded via fileBase.getCustomInputString*(). The finished PDF downloads through result.downloadUrl from temporary storage that rotates after a short retention window. Free.
