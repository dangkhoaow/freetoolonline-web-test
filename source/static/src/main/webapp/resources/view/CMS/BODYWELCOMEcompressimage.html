Last updated: Nov 13, 2024

Image compressor for JPG, PNG, and WebP - pick a compression level, preview the result, and download a smaller file. (Looking to compress a folder or many files at once? Use the ZIP file compressor instead.)

Searching for a file compressor? This is the right tool when your input is a single image; for a folder of files use ZIP File, and for the routing rule see when to ZIP vs when to compress an image.

Compress JPEG online reduces JPG file size for faster sharing and smaller uploads while keeping your photo looking good.

  1. Upload your JPEG image.
  2. Choose a quality level (or enable AI auto-compression).
  3. Click Compress, then download the optimized JPG.

Tip: If you need a specific size, try a couple of quality levels and compare before downloading.
