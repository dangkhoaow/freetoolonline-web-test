PNG to SVG converter traces a raster PNG into a scalable SVG that holds its sharpness at any size.

  1. Upload your PNG file.
  2. Adjust trace settings (colour count, smoothing, threshold) for the artwork type.
  3. Download the converted SVG.

Tip: Logos and flat-colour icons trace cleanly; photographic PNGs almost never do.


NOTE: The output files might display a bit slow because SVG file always contains a lot of information of the image.
